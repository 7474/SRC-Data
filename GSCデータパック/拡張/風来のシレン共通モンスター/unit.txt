
# 初代のモンスターは全て作成しています。
# 壁抜けするユニットは異空間を移動していると考えて宇宙適応をAに。
# 魔蝕虫やオロチといったボスは個別フォルダでの範疇ということで、
# データ化はしていません。

# ユニットはモンスターの系統ごとに並んでいます。
# 強さは下記に記載されているランクを参考にしてください。(Ｅ～Ｓまで)

# 考慮したのは、初代・２・ＧＢ・ＧＢ２の範囲です。
# ※イチゴショートさんの協力でアスカモンスターが作成されました。
# ※MGさんの協力でiアプリ版＋αのモンスターが作成されました。

# 上位モンスターの攻撃力は他の汎用モンスター系と比べて少し高めに設定されています。
# これはシレンモンスターを、ドラクエなどの各種対応型のモンスターとは逆に、
# 遠距離攻撃に乏しい分、強力な近接攻撃をくりだせる猪突猛進型として
# 位置付けているためです。

# 主な製作:ナオヒラ
