
# ポセイドン編の黄金聖闘士の血で蘇った聖衣が新生聖衣。
# ハーデス編のアテナの血で蘇った聖衣が最終聖衣。
# ポセイドン編以前は細かい補修とみなす。
# 一輝は相当レベルということで便宜的に同じ名前に。
# 
# 聖衣によって変わるのは基本的に装甲のみ。火力はセブンセンシズ覚醒でカバー。
# ただし黄金＆神聖衣装備時はセブンセンシズ常時発動とみなし、火力+200の上でブースト無し。
# ＥＮも上昇しますが、消費も多くなります。
# 
# 通常の共闘想定データは最終聖衣まで。
# 黄金聖衣＆神聖衣は限定条件の最終形態、またはイベント形態とみなす。
# 
# シリウス・アイオロス・テティスはアニメ版の技を採用。
