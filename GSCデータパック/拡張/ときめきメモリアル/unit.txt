
# 注釈に「対応クラブ」の指定があるユニットは、
# そのうちいずれかをアイテムとして装備させて登場させるのが原作的には望ましいですが、
# クラブ装備なしでも武装が付くように仕様を変更しました。
# 特にこだわりが無ければ、そのままでも構わないでしょう。

# 各キャラの技はＰＣエンジン版とＰＳ版を基本にしつつ、
# ＳＳ版に登場したものも採用しています。
# その他に、作中の描写を考慮して付けた武装も一部存在します。
# 使役・召喚系の武装は、射撃判定に統一。

# 本来、クラブ奥義は主人公のみ習得するものですが、
# シナリオ終盤での火力不足を補う為の処置として、
# クラブに所属しているヒロインにも、クラブ奥義を付けています。
# 主人公にしたって、三年間部活を続けたという理由だけで習得出来ますし。
# どうしても気になる場合は、ダミー能力の「クラブ奥義取得」をdisableすることで一括封印出来ます。
# 序盤に登場させる場合も、同様です。
