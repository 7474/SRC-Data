# チェリーボムの異名を持つ、爆発物の天才。それでいてビリィよりも身軽とい
# う、なかなか手に追えないキャラクター。
# 赤鼻に「人間武器庫」と恐れられていた(笑)。

# キリエは平気でビリィを巻き込んだ戦い方をするため、また「爆発物」の扱い
# に長けていることを表現するため、敢えて通常武器は取り払い、格闘以外の武
# 装は全てマップ兵器としました。
# 攻撃力の割に必要気力が低めなのは、敵を倒すことによる気力の上昇があまり
# 見込めないためです(マップ兵器で敵を倒しても気力は上がらないので)。
# 装備アイテム数が多いのも、「らしさ」を表現するためです。

# 「爆殺塵」はＣＤドラマ「夕陽に落ちる星」で使っていたわざです。セリフで
# 出てきただけなので正確な字はわかりません。「爆殺」はまず間違いないとし
# ても、「塵」なのか「陣」なのか、或いは別の字なのかはわかりません。正式
# 名称をご存じの方は、ご一報下さると有難いです。
# また、当然ビジュアルによる描写はされていないので、デザインは自分のイメ
# ージです。
