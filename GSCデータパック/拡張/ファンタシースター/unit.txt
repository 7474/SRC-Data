
#・味方　　####################################

#　　コラボは弾数1の必殺技扱いにしました。だいたい一発か二発で壊れる代物ですし。
#　　適宜Disableしてください。普通にやれば使えるのは中盤頃になります。
#
#　　たいていのプレイヤーは、ＨＰ回復能力が追加されるヒールクリスタルを全員に装備させると
#　思われますので、四人ともヒールとの組み合わせで使える技を持たせています。
#
#　アリサ：ラスターバニッシュ
#　ミャウ：ラストフォトン
#　タイロン：ファイヤーウォール
#　ルツ：リターン（祈りで再現）
#
#　　なお、ファイヤーウォールを持たせるくらいならラスターバニッシュを使わせると
#　思いますが、たった四人で二人同じものを持っていても面白くないのでこうしましてます。

#　　ヒールクリスタル、およびエウロパの耳飾り他によるＨＰ回復効果は根性で再現。
#　　全員根性持ってるのは仕様です。

#　　縛り屋です。
#　　原作では攻撃力に防御力、敏捷度、魔力、ＨＰＭＰのすべてにおいて中途半端という微妙な
#　スペックで、麻痺させられるくらいしか特長がありませんでした。
#　　あからさまに低くはないものの全体に微妙で、縛り能力でスロットを狙う感じにしました。

#　　ラコニアンソードは中盤、ラコニアンシールドは終盤までDisable推奨。
#　　なお、ラコニアンシールド入手後は、普通のシールドを封印してください。
#　　アリサのフルネームは最終盤まで明かされません。フルネームが判明するまで愛称変更能力を
#　封印しておいてください。

#　　格闘武器はダガーでもソードでも違和感がないように。
#　　ちょっと長いので、なにかもっといい案があったらご提案いただけるとありがたいです。

#　　ラコニアンシールドがビームシールドなのは、伝説の盾がただのシールドというのも
#　寂しいということで。実際、４人が装備できる盾の中で一番硬いようです。

#　　ラスターバニッシュは、2400くらいなら火力以外にとりえのある人でも持ってるかなと。
#　　一発きりだし射撃低いので大したこともないと見ますが。
