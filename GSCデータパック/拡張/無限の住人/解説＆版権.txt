
～武器解説～「百聞は一見に如かず……」

★万次
# (1)双月：刃が二股に分かれた短い刀剣。序幕の最初でずらっと並んだ時に名が見える
# 解説は無いが、序幕以後もかなりの頻度で用いられている。

# (2)小天狗：双月に似ていますが、刃の一方が鉤状になっている刀剣。７巻末に解説有り
# 木に登ったり、壁をよじ登ったりにもよく使われています。妹守辰政と接続可能

# (3)阿修羅：鎖鎌っぽい刀剣。序幕でしか使ってないかも？序幕の最初に名前が有り

# (4)夜叉：匕首を２つ鎖で繋いだ刀剣。刃付きヌンチャク？序幕の最初に名前が見える

# (5)女蠡：細長く鍔のない刀剣。まともに使われたことはほとんど無い。７巻末に解説有り
# 作者曰く「なんか大工道具とか測量器具だったのでは……」

# (6)男転：鉤がついた刀剣。まともに使われたことはほとんど無い。７巻末に解説有り
# 作者曰く「何これ？」

# (7)妹守辰政：元は妹町の夫、斎藤応為辰政の刀剣だった。２巻末に解説有り
# スターウォーズでダース＝モールが使っていた剣のようにつなげられる。２本１組の日本刀

# (8)無銘：阿修羅にそっくりだが「別物」と断言されている名称不明な刀剣。２巻末に解説有り
# 本編で主に用いている、長い鎖付きの刀剣はこれであるらしい。四道に接続可能

# (9)阿吽：元は逸刀流のハゲた剣士の刀剣だった。二節棍風。２巻末に解説有り

# (10)四道：元は司戸菱安の刀剣だったものを万次が改造したもの。
# 近い武器でいうと……沖縄武器のサイみたいな感じかも？２巻末に解説有り

# (11)烏：元は黒衣鯖人の刀剣だった。やや大きめな手裏剣というイメージ
# ３枚刃の刀と４枚刃の刀で２つ１組。高速回転させて用いる。２巻末に解説有り

★凛

# (2)黄金蟲：殺陣黄金蟲で用いる小型の匕首。２０本以上隠し持っている。２巻末に解説有り

★司戸
# (1)四道：万次の四道の元となった刀剣。司戸は四道とは呼んでないのでしょうが便宜的に……
# ２巻末に解説有り

★黒衣
# (1)烏：後に万次の手に渡る。黒衣はこれで凛の父を３つに斬り捨てている。２巻末に解説有り

★凶
# (1)グラントルコ：仕込み刀が仕込んである突き専用の刀剣。ヨーロッパ辺りの剣らしい。
# ２巻末に解説有り

★閑馬
# (1)井上真改蟲殺：普通の日本刀だが、背に毛が植わっている。匕首も同様。２巻末に解説有り

★槇絵
# (1)春翁：三節槍。舶来品を改造したものらしい。２巻末に解説有り

★天津
# (1)頭椎(斧)：小型ながら重量のある斧。天津は力ではなく技で使いこなす。２巻末に解説有り

# (2)頭椎(剣)：古代風の直剣。未使用なので使い方は不明。２巻末に解説有り

★新夜
# (1)匕首＆日本刀：普通の匕首＆日本刀です

★宇留間
# (1)鉤棹：正式な名称は不明。一度刺さると抜けないような逆棘のついた棹状の武器。
# 鎖と連結できるようになっており、木などを用いて相手を吊り上げる形にして動きを封じる。

# (2)鬼太鼓桴：鉈のような刀剣。斬るというよりは重量で叩きつぶす感じ。７巻末に解説有り

★花田
# (1)燕誅丸：２本１対の突き専門の刀剣。普通の刀の方がましっぽい。７巻末に解説有り

★火瓦
# (1)助拾・雨椿：普通の日本刀。名刀であるらしい。７巻末に解説有り

# (2)神刃：南国風の刀剣。重量で叩きつぶす感じ。７巻末に解説有り

★鬼抜
