# データ作製予定の人物

天津影久
天津, MUG_AnotsuKagehisa.bmp

# 若き逸刀流統主。凛の敵とされる人物であるのだが…
# 逸刀流を率いて浅野道場をはじめとする既存の剣術道場を襲撃しているのは
# 無天一流の体面のために破門にされ狂死した祖父の無念を…というような私怨からではない
# 泰平の世の中で剣士の本質を忘れ、恥や体面に執着する人々への怒りと
# 歪みきった剣の道を「強さを追求する」という本道に戻そうという強い意志からである
# あまりにも純粋な剣士であるがために他の道を選べない悲劇の天才と言えるのかもしれない

川上新夜
新夜, MUG_KawakamiAraya.bmp

# 逸刀流剣士にして面職人。独特の感性の面を作る。
# 凛の母の蹂躙の際に先頭に立つなど、元は悪行三昧で女房子供に逃げられるような男であったが
# 女房が病死し、息子が戻ってきてからは息子と２人の平穏な生活に価値を見いだすようになる
# 凛と万次を斬ることを最後に剣士を辞め、面職人として生活することを決めていたが……
# もっとも戦う理由が明確だった剣士である

百琳(無限の住人)
百琳, MUG_Hyakurin.bmp

# 無骸流の一員。群踏（ぶらはま）という弓の使い手でその毒矢が五本刺されば昇天するらしい
# 現在は髪を金髪に染めた気っ風のよい姐さんだが、元は武家の妻であった。
# 元の夫は剣の道にこだわる人物であり、その意にそぐわない息子を目の前で夫に斬殺される
# 夫を殺した咎で死罪になるはずであったところを拾われて無骸流の一員となる
# 現在は「〜にゃあ」という独特の口調のため、万次に「猫ばばあ」などと呼ばれている

偽一(無限の住人)
偽一, MUG_Giichi.bmp

# 無骸流の一員。サングラスに禿頭の無口な男で二本の大きな傷が頭部にある
# 無愛想だが情には厚いらしい。実力も高く冷静で頭もきれる
# 登場からそれなりに経っているのだが、未だ謎が多い人物である

尸良(無限の住人)
尸良, MUG_Shira.bmp

# 無骸流の一員。「外道」この一言で全てを説明できるような人物である
# 彼の外道具合は凶が妹のように思っていた遊女を嬲りながら惨殺したところからはじまる
# その後も外道を繰り返し、一時は手を組んだ凛もそのあまりもの非道に耐えられず訣別…
# その凛を殺そうとして万次に右腕を切り落とされて、万次を宿敵と認定…
# さらには無骸流を裏切り百琳達を珠崎・鬼抜達に売り払っている…
# 切り落とされた右腕の肉を削ぎ落として自らの骨を鋭利な武器としており
# その時の肉を削ぎ落とす苦痛で白髪化している

宇留間(無限の住人)
宇留間, MUG_Uruma.bmp

# 逸刀流剣士。手形を手に入れるために百琳が天津の名を偽って出した偽書簡により、
# 火瓦・花田と共に加賀へ向かう。花田とは腐れ縁な感じ。常に煙管をくわえている。
# 賭事に弱く、登場時も賭けに負けて花田の荷物を持っていた。
# 逸刀流の唯一の信条『一対一』をも違えた花田との連携プレイを見せるも万次に返り討ちに

花田(無限の住人)
花田, MUG_Hanada.bmp

# 逸刀流剣士。手形を手に入れるために百琳が天津の名を偽って出した偽書簡により、
# 火瓦・宇留間と共に加賀へ向かう。宇留間とは腐れ縁な感じ。暑さが苦手らしく、半裸で登場
# 他にも左右の形の違う色眼鏡、麦藁帽子など時代を先取りし過ぎなファッションが印象的
# 逸刀流の唯一の信条『一対一』をも違えた宇留間との連携プレイを見せるも万次に返り討ちに

火瓦(無限の住人)
火瓦, MUG_Higa.bmp

# 逸刀流剣士。手形を手に入れるために百琳が天津の名を偽って出した偽書簡により、
# 宇留間・花田と共に加賀へ向かう。宇留間と花田のお目付役な感じ。
# ターバンのような布を頭に巻き、背中に２本の刀を差している。
# 宇留間・花田と戦いでボロボロになった万次と対決し片手を失いながらも勝利。
# 単純に実力だけをみた場合でも、かなりの実力を持っているものと思われる。
# 観察眼が優れており頭も良さそうだが、不老不死に異常な執着をみせ、思想的には少々問題あり
# 万次に追いついてきた百琳と真理路によって殺される。

鬼抜(無限の住人)
鬼抜, MUG_Kinuka.bmp

# 逸刀流剣士。かなりの大男で、刀も巨大な野太刀を用いる
# 百琳が殺し損ねた剣士、珠崎の口より無骸流の存在を知り珠崎と共に百琳を拉致する。
# 偽一との戦闘の際に左手一本を犠牲にして勝ちを狙うなど、眼力・覚悟ともに優れた剣士
# 結局は一矢も報いることもなく破れることになるが、最後まで誇りを失わぬ態度は立派であった
