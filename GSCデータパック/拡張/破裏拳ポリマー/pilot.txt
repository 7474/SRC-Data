# がかぶり、「転身」した姿。
# その全身は重化合物ポリマーで覆われ、銃弾をも弾き返す耐久性を与える。またフィー
# ルド推進と呼ばれる磁力推進により、超人的なパワーを発揮する。
# そして重化合物ポリマーの配列を変更する事により、５種類のポリマーマシンへメカ転
# 身を行え、あらゆる環境で高い戦闘能力を発揮できる。

# そんなポリマーにも弱点は無いわけではない。ポリマースーツは通気性が無い為、転身
# したまま４６分１秒を過ぎると皮膚呼吸ができなくなり死んでしまう。
# また強力な電流を受けるとフィールド推進が封じられ、非常な苦痛を味わう事になる。
# さらに、ポリメットが作動可能な限界温度はマイナス５０度。これを下回ると転身を行
# えず、例え転身した後であっても影響を完全に免れる事は出来ず、転身が解除されてし
# まう事もある。
