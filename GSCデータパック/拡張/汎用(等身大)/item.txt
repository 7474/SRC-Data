# 以下のアイテムは攻撃力に乏しい銃火器をメインに戦うキャラクターが、
# 後半戦に対応するために用意しています。

# これらのアイテムを装備することを想定したユニットを制作するときは、
# 特殊能力「武器クラス=銃火器」「装備個所=腕」を付けて下さい。

# 手榴弾系以外は一つしか装備できないので、
# いわゆる「アサルトバスター状態」「フルウェポン状態」になることはあまりないでしょう。

# ザコに持たせるのはニューナンプ、密造トカレフ程度にしておくのが無難でしょう。
# もっと強力な装備が欲しい時は、各作品別フォルダの銃火器等を使ってください。
