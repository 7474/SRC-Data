## include.eve ##

ＨＰ条件解説処理:
Local M_HP
If 対象ユニット使用武器番号 > 0 Then
  M_HP = Info(ユニット,対象ユニットＩＤ,武器,対象ユニット使用武器番号,属性レベル,Args(1))
ElseIf 対象ユニット使用アビリティ番号 > 0 Then
  M_HP = Info(ユニット,対象ユニットＩＤ,アビリティ,対象ユニット使用アビリティ番号,属性レベル,Args(1))
EndIf
M_HP = IIf(Args(2) = ＨＰ,(Int(Info(対象ユニットＩＤ,最大ＨＰ) * (M_HP / 10))),(M_HP * 10))
Return M_HP
