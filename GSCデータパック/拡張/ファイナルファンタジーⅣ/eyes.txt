##### ファイナルファンタジーⅣ:「警報」原作再現インクルード #####
# アイズ・サーチャーの「警報」の仕様を、
# 原作同様、攻撃されるとモンスターを召喚する仕様に変更するインクルです。
#
# 警報が発動すると、
# アイズはキマイラ、フレイムドッグ、ストーンゴーレム、ナーガのいずれかを、
# サーチャーは機械兵、鉄騎兵、巨人兵、機械竜のいずれかを呼び出し、
# ビームによる迎撃モードに移行します。
#
# 別のユニットを呼び出したい、警報の発動条件を変えたい、
# 同じマップでもアイズ毎に違うユニットを呼び出したいなどの場合は、
# 自己責任で、書き替えるなり自作するなりの対応をしてください。

# 使用時にはunit.txtの該当箇所を以下のように書き替えてください。
# パイロットは、召喚するユニットのパイロットを乗せてください。

### unit.txt ###
# アイズ(ＦＦ４)
# アイズ, (ＦＦ４), 1, 2
# 空, 3, SS, 2800, 65
# 特殊能力
# 無効化=魔
# 弱点=機式
# 性格変更=機械
# ダミー特殊能力=警報 攻撃されると警報音を鳴らし、モンスターを呼び出す。
# パイロット愛称=アイズ
# 4000, 50, 500, 40
# AACA, FF04_EyesU.bmp
# 警報(FF04),                        0, 1, 5, -99,  -,   -,   -, AAAA,  +0, 音反|脱L0 <!警報済>
# ビーム,                         1200, 1, 4, +40,  -,  10,   -, AAAA,  +0, Ｂ <警報済>
#
# サーチャー(ＦＦ４)
# サーチャー, (ＦＦ４), 1, 2
# 空, 3, SS, 2800, 90
# 特殊能力
# 弱点=雷機式
# 性格変更=機械
# ダミー特殊能力=警報 攻撃されると警報音を鳴らし、モンスターを呼び出す。
# パイロット愛称=サーチャー
# 4000, 80, 1300, 75
# AACA, FF04_SercherU.bmp
# 警報(FF04),                        0, 1, 5, -99,  -,   -,   -, AAAA,  +0, 音反|脱L0 <!警報済>
# ビーム,                         1200, 1, 4, +40,  -,  10,   -, AAAA,  +0, Ｂ <警報済>

### 使用変数 ###
# FF04_警報召喚ユニット:警報で召喚するユニット名を格納します

*使用後 味方 警報(FF04):
*使用後 ＮＰＣ 警報(FF04):
*使用後 敵 警報(FF04):
*使用後 中立 警報(FF04):

	Switch Unit(対象ユニットＩＤ)

	Case アイズ(ＦＦ４)
		Switch Random(4)
		Case 1
			Set FF04_警報召喚ユニット キマイラ(ＦＦ４)
		Case 2
			Set FF04_警報召喚ユニット フレイムドッグ(ＦＦ４)
		Case 3
			Set FF04_警報召喚ユニット ストーンゴーレム(ＦＦ４)
		Case 4
			Set FF04_警報召喚ユニット ナーガ(ＦＦ４)
		EndSw

	Case サーチャー(ＦＦ４)
		Switch Random(4)
		Case 1
			Set FF04_警報召喚ユニット 機械兵(ＦＦ４)
		Case 2
			Set FF04_警報召喚ユニット 鉄騎兵(ＦＦ４)
		Case 3
			Set FF04_警報召喚ユニット 巨人兵(ＦＦ４)
		Case 4
			Set FF04_警報召喚ユニット 機械竜(ＦＦ４)
		EndSw

	Case Else
		Exit

	EndSw

	SetSkill (対象パイロット) 警報済 -1
	Create Party(対象ユニットＩＤ) (FF04_警報召喚ユニット) Rank(対象ユニットＩＤ) Pilot(対象ユニットＩＤ) Level(対象パイロット) X(対象ユニットＩＤ) Y(対象ユニットＩＤ)
Exit
