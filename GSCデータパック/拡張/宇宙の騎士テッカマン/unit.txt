#   テッカマンの使用する鞭

#
# ・テックスパイク
#   両脚のかかとにある、切れ味鋭い拍車
#   武器としてのほか、ペガスの前進を促す合図としても使用される
#
# ・テックランサー
#   テッカマンの武装でもっとも特徴的な物
#   グリップの両端から刀剣状の刃が飛び出し、ランサーとして使う
#   剣先を撃ち出す事も可能
#   攻撃・防御と様々に多用する
#
# ・ビームブレイク
#   テックランサーの両剣先を飛ばして敵を挟み込み、強力な電磁波を浴びせる必殺技
#
# ・大回転テックランサー
#   回転させて放り投げたテックランサーにボルテッカをぶつけ、エネルギーを拡散照射
# する必殺技
#   エネルギー下の敵を一網打尽にすることが可能
#
# ・ボルテッカ
#   額のマークにエネルギーを乗せて撃ち出す必殺技
#   威力は絶大だが、発射後の額の穴がその瞬間の弱点となる上、体力の消耗も激しい諸
# 刃の剣
#
# ・ボルテッカ三段返し
#   ボルテッカを三連射する技
