
#　　共通能力として以下を持ちます。
#　　原作でのブレイク状態の表現になります。
#
#　クエスター
#　抵抗力Lv3=非表示 (瀕死)
#　ＥＮ消費減少Lv2=クエスター (瀕死)
#　クエスター=解説 瀕死状態のとき、特殊効果発動率を30%低下させ、ＥＮ消費量が20%減少する。 <瀕死>
#
#　サクセサー
#　抵抗力Lv3=非表示 (瀕死)
#　ＥＮ消費減少Lv2=サクセサー (瀕死)
#　サクセサー=解説 瀕死状態のとき、特殊効果発動率を30%低下させ、ＥＮ消費量が20%減少する。 <瀕死>
#
#　アインヘリアル
#　弱点=神
#　抵抗力Lv3=非表示 (瀕死)
#　ＥＮ消費減少Lv2=アインヘリアル (瀕死)
#　アインヘリアル=解説 瀕死状態のとき、特殊効果発動率を30%低下させ、ＥＮ消費量が20%減少する。 <瀕死>(!不老不滅)
#　アインヘリアル=解説 瀕死状態のとき、ＥＮ消費量が20%減少する。 <瀕死>(不老不滅)

#　　範囲攻撃は主に命中とＣＴの高い攻撃で、たまにマップ兵器になっています。
#　　シーン攻撃は基本的にマップ兵器です。
#　　原作は属性ごとに防御修正の差がありますが、ＳＲＣでは装甲にすべて含まれます。
#　ですので、〈炎〉〈氷〉〈雷〉〈光〉〈闇〉は攻撃力を上げ気味にしています。
#　　また、〈神〉属性の攻撃は魔法武器もしくは無属性にしています。

#　　加護は原作的にはすべて弾数1なのですが、こだわると大変なので気にせず
#　バリアや阻止、能力強化などで再現しているものもあります。
#　《オーディン》は基本加護の１つなので迷いましたが、思いきってオミットしました。
#　その代わり、ＥＮや攻撃力、他の加護の効果などを少し底上げしています。
#　《トール》《ヘル》《ネルガル》あたりはけっこう他者がけで使うことも多いんですが、
#　アビリティにすると煩雑になるので本人が使った場合を想定してデータ化しています。
