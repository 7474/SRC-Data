## ダンテ＆デビルアームズ  ##

######################################################################
#【悪魔狩人ダンテ】
# 　現代社会の暗部に通じ、金ずくで危険な依頼を引き受ける凄腕の便利屋。
# ただし、依頼が気に入らなければいくら金を積まれても引き受けない変わり
# 者。
# 　その正体は、伝説の魔剣士スパーダと人間の間に生まれた魔人であり、血
# に飢えた地獄の魔獣を次々と屠る“悪魔も泣き出す”デビルハンターである。
#
#・フォースエッジ
# 　ダンテの父であり、伝説の魔剣士であるスパーダの大剣。
# 　大きくて肉厚な剣だが、ほかにこれといった特徴は無い。
# 
#・スパーダ
# 　ダンテとバージルの持つアミュレットが合わさったパーフェクトアミュレッ
# トにより現れた、フォースエッジの真の姿。
# 　巨大な刀身を持ち、攻撃にあわせて槍や鎌のように変化する。
#
#・アラストル
# 　苦痛に悶えた女性のレリーフに突き立っていた大剣。手に入れる時、その
# 人物の心臓を欲する。ダンテも入手時に心臓を貫かれた。
# 　電撃を帯び、持つ者にスピードと飛翔能力を与える。
#
#・イフリート
# 　コロシアムの中央に納められていた籠手。気性が荒く、手に入れる時その
# 人物を地獄の業火で焼き尽くす。ダンテも全身を焼かれた。
# 　火焔を帯び、持つ者の力を増大させる。
# 
#・エボニー＆アイボリー
# 　ダンテ(トニー)のために作られた、世界でただ一対の双子の拳銃。ダンテ
# の驚異的な速射に対応できる能力がある。
# 　エボニーが精密射撃用に作られた左手の黒い銃で、アイボリーが速射性能
# を重視した右手の白い銃。形状のベースはコルト・ガバメントと思われるが、
# 原型とはかけ離れた巨大な銃になっている。
# 　スライド表面に『BY .45 ART WARKS　FOR TONY REDGRAVE』と書かれている。
#
#・ショットガン　グレネードガン
# 　この二つの銃は、人間界で製作された一般的なもの。
# 　ショットガンは城の図書室のようなところの机の上に置いてあり、グレネー
# ドガンは過去に城に侵入したと思われる戦士の亡骸が持っていた。
#
#・ナイトメアβ
# 　魔界で作られた兵器。その名前の通りナイトメアを小型にしたような形状
# をしており、反射する特性を持つビームを放つ。
# 　硬度も高いらしく、装備したまま腕で攻撃するとダメージが増加する。


## ヒロイン ##

######################################################################
#【トリッシュ】
# 　月夜の晩、バイクでダンテの事務所『Devil May Cry』に突撃し、ダンテ
# に襲い掛かった謎の美女。その顔は、ダンテの亡き母エヴァによく似ている。
# 　彼女の依頼でダンテはマレット島で起こっている異変を知り、そこに赴く
# ことになった。
# 　実はそれがスパーダの息子であるダンテを抹殺するためのムンドゥスの企
# みであり、トリッシュはそのために創られた存在だった。
# 　一時はムンドゥスの手により殺されるが、その後復活。ダンテと共に便利
# 屋『Devil Never Cry』を開く。

## トニー・レッドグレイブ ##

######################################################################
#【トニー・レッドグレイブ】
# 　クール＆タフを標榜する、実力派の便利屋。悪魔よけのお守りと称するア
# クセサリーを大量に身に着けている。
# 「殺しなどダサい」と昔気質の便利屋からすれば甘いことをいいながら、そ
# の実力を持って周辺の便利屋業界のしきたりを一変させた。また、それに反
# 発してトニーを排除しようとしたマフィアは、ことごとく全滅の憂き目にあ
# うはめになる。
# 　その容姿や、本人も気があれば足繁く通うだけの甲斐性もあり、女性経験
# は豊富。しかし、「ママ」という寝言を聞かれ、マザコン疑惑をかけられて
# フラレることも多い。ただ、それをネタにして引き止めておこうとする女性
# もいる。
# 　グルーとは関係が深く、家族ぐるみの付き合いをさせてもらっている。ま
# た、ニールとも家族のような関係を築いている。
#
# 　実はトニー・レッドグレイブは偽名である。子供の頃、悪魔に襲撃された
# 時に「今はその名を隠し、目を眩ませ、逃げ延びよ」と、父の形見である剣
# に語りかけられたためだ。
# 　そして後に、グルーとその娘のジェシカ、そしてニールなどを悪魔に殺さ
# れたことにより、甘えを捨てて真名を名乗る決意をする。
