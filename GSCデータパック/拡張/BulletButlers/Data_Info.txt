# 「銃弾切り払い」について
#
# 作中、「銃弾切り払い」をする描写が
# セルマ・雪・レイスなどに見受けられましたが無しになっています。
# 理由は以下の通り。
# Ａ：「悪鬼喰」を切り払うのはフォルダ的におかしい。
# Ｂ：「銃領支配」で「悪鬼喰」を止める再現は必要だろう。
# 
# 「悪鬼喰」が他所フォルダでは切り払われたり、
# 例えば「電磁銃」(BLACKCAT)を「銃領支配」が止めれないなどはあるのですが
# そこら辺はデータを扱う側で調整していただきたいなと。
