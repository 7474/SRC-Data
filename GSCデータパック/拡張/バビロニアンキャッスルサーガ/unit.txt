####　敵の呪文について　####
#　　主な呪文の使い手であるメイジが、突然テレポートしてきて呪文を
#　放つという特性を持つため、基本的にＰ兵器にしています。
#
#・ドルアーガの塔とドルアーガの迷宮
#　　ホワイトスペル：単純に2Pです。
#　　レッドスペル：炎がしばらく残るという特性から2-2Pで高命中としました。
#　　ブルースペル：壁を破壊する効果から、威力が他のスペルより高めです。
#　　グリーンスペル：壁を通り抜ける効果を3-3P（ボス格は2-3P）で表現。
#　　　　　　　　　　また、ギルに近づいてこない敵は間属性になっています。
#　　パープルスペル：攻撃効果はグリーンスペルと同じなので同様に。
#　　イエロースペル：使うのがメイジ系ではないので普通に1-4射程にしています。
#　　　　　　　　　　壁を突き抜けて吹き飛ばすので転のほうが正しいかもですが、
#　　　　　　　　　　転属性はクリティカルしないと発動しないんですよね。
#
#・イシターの復活
#　　メイジ系は壁を通過しないスペルを1-2P、壁を通過するスペルは
#　1-5射程としています。
#　　スライム系のスペルは1-3射程です。
