#    デフォルメしてある。
#  ○ライオンハート○
#    スコールの最強武器です。
#  ○ブラスティングゾーン○
#    剣から発した巨大なエネルギーの奔流（衛星軌道上から確認できるサイズ）を創り出し
#    敵目掛けて振り下ろす。

#    スコールの最終奥義です。連続で斬り付けます。
