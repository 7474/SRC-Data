

##  主要登場人物  ##  =============================================  ##


シド＝クレイマー(ＦＦ８)

#  バラム・ガーデンの学園長。
#  温和な性格をしており、教育者としては相応しいといえる。
#  妻イデアと共に、ガーデンを作り、SeeDを育てた。
#
#  『SeeDはなあ、未来の為にまかれたタネだ！
#    その未来が今ナンダヨ！それはあんただってわかってるだろうが！』

レイン, FF08_Raine.bmp

#  怪我して倒れていたラグナを助けて、知り合うことになる。


エルオーネ(ＦＦ８)
エルオーネ, FF08_Ellione.bmp
#  レインの家の隣に住む子供。戦争で両親を失った為、レインに育てられる。
#  エスタの幼女狩りにあい、誘拐される。その後、ラグナによって救助されるが……
#
#  自分と会ったことのある人物の精神を過去に送る能力を持っている為、それを欲した
#  未来の魔女アルティミシアに狙われてしまう。
#  ＦＦⅧの世界で、魔女以外の人間が魔法を使えるのは、
#  彼女の能力がオダイン博士によって研究された為である。
#  作中、最も重要なポジションに位置するといえる。
#
#  『頼れるのはあなた達だけなの……』
