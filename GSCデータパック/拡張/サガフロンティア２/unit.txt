
#　　クヴェルは、基本的に手に入るシナリオに登場するキャラのうち誰かに持たせてあります。
#　　アニマは、術士キャラ以外は初期装備＋持っているクヴェルのアニマということにして
#　います。術士は、術がメインだしってことでその辺の制限とかはあんまり考えてません。

#　　装備固有の技ですが、ある程度特殊なものか、合成術が使えるもの以外はオミットです。
#　　サブウェポンはアニマを引き出すために持っているだけという扱いで、データには含めて
#　いません。ただし、弓がメインウェポンである場合を除きます。

#　　アニマと金属の関係というのは世界観的に重要な要素です。なんで、金属鎧を装備したキャ
#　ラクターは魔属性に対するアーマーを持ちます。
#　　射撃がその分低くなっていますが……意味があるのはパトリックくらいですかね。

#　　技：物理属性＆格闘攻撃
#　　合成術：魔術属性＆射撃攻撃
#　　術技：魔属性＆射撃攻撃
