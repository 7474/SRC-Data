#…スペースガチョビンスーツ
#
#板垣総理が提唱する、世界人類が皆ガチョビンになることにより
#世界人類のかかえるすべての問題を解決する究極計画

#                「人類皆ガチョビン化計画」
#
#の最終形態で、来たるべき宇宙時代には世界人類全員がスペース
#ガチョビンスーツを装備する事により、宇宙でも自在に生活が可能
#になるのである。
#
#スペースガチョビンスーツは兵器としても優秀であり、その火力は
#「ガンダム三号機と００４を足して２で割って牙流忍術をかけた
#ほど」「全パワーを使えば九龍半島が無くなる」との事。
#
#しかしスペースガチョビンスーツにも致命的な弱点が存在する。
#それは頭部の角である。この角が取れてしまうと、ただのガチャピ
#ンと化してしまい「世間が黙っていても、ピエール瀧が黙っちゃ
#いない」との事で、その時点で勝負を放棄しなければけならないの
#である。また、宇宙用として設計され、完全気密性を誇るスペース
#ガチョビンスーツであるが、その構造が最終的に板垣総理の命奪う
#こととなってしまったのは皮肉な話である。
#
#
#…武装解説…
#
#…屁…
#総理の愛犬「外務大臣」を死に至らしめるほどの臭さを誇るオナラ。
#それを十年間我慢し続けたものを一気に放出する板垣総理の最後の
#手段である。しかし、スペースガチョビンスーツが完全気密性であ
#る事を忘れて使用してはならない。

#核兵器級の威力を持つ驚異のロケットパンチ。
#両腕に１発ずつ、計２発が装備されている。

#…キャッツカード…
#キャッツアイのアレ。
#「敏夫さん、瞳をよろしくね」
#
#…トロフィー噴射…
#尻の穴にトロフィーを突き刺し、オナラの爆風で発射する技。
#変態と勘違いされる危機にあった吉崎を救った技でもある。
#この技が下の「ギャリック砲」に発展する。
#
#…奈良カッター…
#奈良の髪の毛に偽装された十字型のカッター。
#奈良いわく「奈良カッターは岩をも砕く」だそうである。
#
#…奈良ブーメラン…
#奈良カッターを投げつけ、ブーメランにする技。
#「奈良ブーメランは狙った獲物は逃さない」という話だが、
#まともに相手に命中した事は一度も無い。
#
#…奈良スクリュー…
#相手に浣腸を決め、そこから奈良自身が超高速回転をし、
#食らった相手は天国と地獄を同時に味わう事になる奈良の必殺技。
#付加効果として混乱のステータス異常を引き起こす事がある。
#
#…吉六会奥義・奈良づくし…
#相手の体に裸で抱きつき、チ◯ポをこすりつけるという恐怖を
#体験させ、いずれは死に至らしめる、もしくは廃人にしてしまう
#という恐るべき奥義。
#ＳＲＣ上では攻撃力はさっぱりだが、様々な状態異常を引き起こす
#というデータになっている。
#
#なお、チ◯ポの硬質度は気力での表現をしてあります。そのため、
#精神コマンド「気合」を使用すると奈良独自のチ◯ポの硬化作業である
#「エビの脱皮」を開始するように設定しました。
#
#…ギャリック砲…
#尻の穴に奈良カッターをセットし、オナラの爆風で奈良カッター
#を発射する奈良の最後の技。かめはめ波である。
