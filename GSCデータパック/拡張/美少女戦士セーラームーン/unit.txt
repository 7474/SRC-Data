# 終盤：セーラームーン　※変形解禁。ハレーションとエスカレーション両取りも可。

# ●パターン２：アニメ全作を網羅
# 序盤１：セーラームーン(初期)
# 序盤２：セーラームーン　※変形封印

# 中盤２：スーパーセーラームーン(ＳＳ)

##-------------------ムーン-------------------##
# 突撃を活用していい場所を確保できる、砲台役
# 大火力の光浄武装と、強力な治癒アビリティが売り。
# ムーンフリスビーやティアラブーメランは、ティアラアクションを適宜リネームしてください。

# ●原作準拠する場合の武装＆ユニット強化手順
# 無印前半	：プリンセスであることを知らない頃です。
#				ユニット「セーラームーン(初期)」
#				Disable セーラームーン(初期) ＭＨエスカレーション(初期)
#				Disable セーラームーン(初期) ムーンヒーリング
# 無印中盤	：過去の記憶を取り戻した状態です。
#				ユニット「セーラームーン(初期)」
#				Enable セーラームーン(初期) ＭＨエスカレーション(初期)
#				Enable セーラームーン(初期) ムーンヒーリング
# 無印終盤	：キーアイテムである「銀水晶」を奪還した状態です。
#				ユニット「セーラームーン」
#				Disable セーラームーン Ｍプリンセスハレーション
#				Disable セーラームーン 幻の銀水晶
# Ｒ		：「銀水晶」を失いパワーダウンします。
#				ユニット「セーラームーン」
#				Disable セーラームーン ＭＨエスカレーション
#				Enable セーラームーン Ｍプリンセスハレーション
# （Ｓ		：セーラームーン(Ｓ)にUpgrade）

# セーラームーン(初期) は主にアニメーションの関係で分けたユニットです。
# ムーンスティックに銀水晶がないアニメとなっています。
# 通常のシナリオでは飛ばしてもかまわないでしょう。

# Ｒの時期は設定的にパワーダウンしていますが、データ上は燃費の優位を得ます。
# バランス的に辛い場合は早めにセーラームーン(Ｓ)にUpgradeしてください。
# また、「ヒーリングエスカレーション」を同時使用可能にしてもよいでしょう。

# 変身形態「プリンセス・セレニティ」は、
# イベント戦での使用を想定したものではありますが、
# 無印ユニットで終盤に乗り込むための強化要素としても利用可能です。
# ハイパーモード先はエターナルセーラームーンと同じくらいの戦闘力があります。
