## ダンテ＆デビルアームズ  ##

######################################################################
#【悪魔狩人ダンテ】
# 　現代社会の暗部に通じ、金ずくで危険な依頼を引き受ける凄腕の便利屋。
# ただし、依頼が気に入らなければいくら金を積まれても引き受けない変わり
# 者。
# 　その正体は、伝説の魔剣士スパーダと人間の間に生まれた魔人であり、血
# に飢えた地獄の魔獣を次々と屠る“悪魔も泣き出す”デビルハンターである。
#
# ・リベリオン
# 　ダンテの所有する身の丈ほどもある大剣。魔力を帯びると形状が変化する。
# 　汎用性に富み、あらゆる局面で威力を発揮する。
#
# ・三氷棍ケルベロス
# 　テメンニグルの門番であるケルベロスに力を認められると入手できる。
# 　通常の三節棍とは異なった特殊な形状をしており、常に冷気を纏っている。
# 　スピードに優れた武装であり、魔人化時に最大限発揮される。
#
# ・炎風剣アグニ＆ルドラ
# 　双子の悪魔であるアグニとルドラに力を認められると入手できる。
# 　鋸のような形状の二本の剣。アグニは炎、ルドラは風を纏っている。
#
# ・雷刃ネヴァン
# 　ネヴァンに力を認められると入手できる。
# 　ギターのような形状であり、雷を生み出すことが出来る。また、鎌のよう
# な形状に変形することも出来る。
#
# ・閃光装具ベオウルフ
# 　ベオウルフの力を宿した籠手と足の防具。
# 　強力な光を纏っており、爆発的な攻撃力の手技を使う。
#
# ・エボニー＆アイボリー
# 　ダンテのもつ巨大な二挺拳銃。
# 　ダンテのマシンガンに匹敵する連射能力についていくため、通常では考え
# られないほど肉厚な銃身が特徴。
#
# ・アルテミス
# 　魔界の銃火器。魔力の矢を発射し、標的に対して攻撃を行う。
# 　強化により複数の矢を生み出すことが可能。
#
# ・ショットガン　スパイラル
# 　一般的な銃火器。
# 　スパイラルは巨大なライフルの名称。
#
# ・カリーナ＝アン
# 　レディの愛用するミサイルランチャー。様々なギミックが施されている。
# 　その名はレディの母の名でもある。
#
# ・トリックスター・スタイル
# 　ダンテの特殊な戦闘技術。
# 　高い身体能力と魔力を活かした様々な体術を行使することができる。
#
# ・ソードマスター・スタイル
# 　ダンテの特殊な戦闘技術。
# 　近接武器の能力を引き出し、様々な技を行使することができる。
#
# ・ガンスリンガー・スタイル
# 　ダンテの特殊な戦闘技術。
# 　遠距離武器の能力引き出し、様々な技を行使することができる。
#
# ・ロイヤルガード・スタイル
# 　ダンテの特殊な戦闘技術。
# 　相手の攻撃に対してガードを行うことが出来る。また、ガードによって蓄
# 積したエネルギーを放出して攻撃することも可能。
#
# ・クイックシルバー・スタイル
# 　ゲリュオンに力を認められると入手できる戦闘技能。
# 　周囲の時を止めることができるようになる。
#
# ・ドッペルゲンガー・スタイル
# 　自らとの戦いに打克つと入手できる戦闘技能。
# 　ドッペルゲンガーを操り、二重の攻撃を可能にする。


## 魔剣士バージル  ##

######################################################################
#【魔剣士バージル】
# 　ダンテの双子の兄であり、悪魔として生きることを選んだ魔剣士。
# 　力こそが正義であると信じ、非常な手段を使うことを厭わない。
#
# ・閻魔刀
# 　バージルの持つ父の形見の刀。「やまと」と読む。
#
# ・閃光装具ベオウルフ
# 　視力を失い、臭いのみでダンテを追跡してきたベオウルフを打ち破り、力
# ずくで入手した。
# 　爆発的攻撃力を持つ足技を使う。
#
# ・フォースエッジ
# 　伝説の魔剣士であり父であるスパーダの愛剣。圧倒的な魔力を誇り、その
# 力は並の人間の制御できるものではない。
# 　バージルはこのフォースエッジと閻魔刀の二刀流で戦闘を行う。


## ヒロイン ##

######################################################################
#【デビルハンター・レディ】
# 　悪魔を憎み、悪魔を全て狩るために戦い続けるデビルハンター。
# 　高い身体能力は努力によって身につけたものだが、伝説の魔剣士の血を引
# くダンテとバージルには全く歯が立たない。
# 　ちなみに、本来の名前は「メアリ」と言うが、本人は既にその名前を捨て
# ているため、「レディ」とダンテに呼ばれて以来この名前で通している。


## ボス ##

######################################################################
#【ケルベロス】
# 　テメンニグルを守る三つ首の犬の姿をした門番。通常イメージするケルベ
# ロスと違い、体中を氷で覆った姿をしており、攻撃も冷気を用いる。
# 　力なきものの進入は決して許さないが、いったん力を認めれば自身の力を
# 貸すという義理堅い面も見せる。

######################################################################
#【ギガピード】
# 　元々は魔界の巨大な生物だが、テメンニグルが生み出した空間のゆがみか
# ら人間界に現れてしまった。
# 　体中から雷を発生させ、縄張りに侵入した敵を容赦なく攻撃する。

######################################################################
#【アグニ＆ルドラ】
# 　テメンニグル内の扉を守ることを命ぜられている双子の悪魔。アグニが兄、
# ルドラが弟。
# 　表面上強靭な肉体を持つように見えるが、本体は剣。現状の肉体に満足し
# ておらず、より強く自分達を操ってくれる体を求めている。

######################################################################
#【リバイアサン】
# 　ギガピードよりもさらに巨大な魔界の生物。テメンニグルの回りをグルグ
# ルと回っていたところ、目の前に飛び出してきたダンテを餌だと思い飲み込
# んでしまう。
# 　体の中は迷路のようになっており、魔界と繋がっているため悪魔も存在す
# る小さな魔界となっている。
# 　特に心臓と肺は独自に防衛機能まで備えており、危害を加えるものを容赦
# なく攻撃する。

######################################################################
#【ネヴァン】
# 　過去にスパーダに封印された悪魔。妖艶な魅力で淫蕩の道へと人を堕落さ
# せてきた。
# 　影と電撃を纏ったコウモリによる攻撃を得意としている。

######################################################################
#【ベオウルフ】
# 　過去にスパーダに封印された悪魔。それ以来、復讐の機会を虎視眈々と待
# ち続けていた。
# 　嗅覚に優れているらしく、ダンテとバージルに流れるスパーダの血の臭い
# を嗅ぎ取っていた。
# 　巨大な体と強大な力を持ち、他の悪魔を圧倒するパワフルな攻撃を行う。

######################################################################
#【ゲリュオン】
# 　数多の英雄を乗せて戦場を駆けてきた名馬。しかし、魔界の瘴気を浴びた
# ことで悪魔となってしまった。
# 　時を操る能力と、数多の英雄が用いてきた武器を使った戦いをする。

######################################################################
#【ドッペルゲンガー】
# 　自分と同じ姿をしており、それを見たものには死が訪れるといわれる悪魔。
# 　その正体は特殊な光によって生み出された影であり、再びその光を浴びな
# い限り、全ての攻撃を無効化してしまう。

######################################################################
#【アーカム】
# 　悪魔になることを望み、非道な悪行を重ねてきた人間。
# 　バージルと共闘するも、様々な秘密を持ちつつ、いつでも相手を出し抜け
# るよう虎視眈々と狙っている。


## 悪魔 ##

######################################################################
#【７ヘルズ】
# 　七つの大罪の名を冠した悪魔。それぞれに対応する罪を犯した人間の魂を
# 地獄で攻め続ける。
#
# ・ヘル＝プライド【強欲】
# 　砂を媒介に現世に現れた悪魔。
# ・ヘル＝エンヴィ【嫉妬】
# 　魔界と繋がっているリバイアサンの体内に現れる悪魔。
# ・ヘル＝レイス【憤激】　
# 　魔獣の心臓を持ち、それを爆発させることで攻撃を行う。
# ・ヘル＝スロース【怠惰】
# 　一瞬砂に戻りワープする能力を持つ。
# ・ヘル＝グリード【強欲】
# 　手にした棺を振り回し、他の７ヘルズを召喚して攻撃を行う。
# ・ヘル＝グラトニー【暴食】
# 　大気中の砂を吸収し、圧縮して吐き出すことで衝撃波を生み出す。
# ・ヘル＝レイス【色欲】
# 　７ヘルズの中で最も素早く、不意打ちを得意とする。

######################################################################
#【アビス】
# 　魔界の深遠に生息する高位の悪魔。戦争の際には魔界の先兵として兵役に
# つく。
# 　姿形は７ヘルズに似ているものの、あらゆる能力が桁違いに高い。

######################################################################
#【ヘル＝バンガード】
# 　７ヘルズを統括する高位の悪魔。
# 　７ヘルズのボスであり、部下の敵を討つために魔力で形成された鎌を振る。

######################################################################
#【エニグマ】
# 　魔術により動き出した石像。
# 　手にした弓から魔力の矢を打ち出す魔弾の射手。

######################################################################
#【ブラッドゴイル】
# 　魔力を込めて作られた石像に悪魔の血を浴びせたことにより生まれた液状
# の悪魔。
# 　ほとんどの旧態依然とした物理攻撃を無効化するが、銃火器など近代兵器
# に対しては防御力を持たない。

######################################################################
#【デュラハン】
# 　悪魔に取り付かれた動く鎧。
# 　正面の盾はあらゆる攻撃を無効化してしまうため、背面から攻撃する必要
# がある。

######################################################################
#【アルケニー】【クイーンアルケニー】
# 　女性の魂が魔界の瘴気により悪魔となった姿。
# 　長期間生き延びたアルケニーはクイーンとなり、体の色素も変質する。

######################################################################
#【フォールン】
# 　かつては天上界に存在したとも言われる、人に甘言をささやき堕落させる
# 悪魔。
# 　神々しい姿と純白の翼を持つが、翼の下の腹部には醜い顔が隠されている。

######################################################################
#【ソウルイーター】
# 　殺された悪魔の魂が集合し、別種の悪魔となった。
# 　生者に取り憑き、魂を食らう。

######################################################################
#【ダムドチェス】
# 　魔界で製造された、魔界のチェス。魔界のチェスは凶暴です。
#
# ・ダムドポーン
# 　もっとも弱いが、プロモーションという別の駒に変化する能力を持つ。
# ・ダムドルーク
# 　高い機動力と攻撃力を持つ。
# ・ダムドビショップ
# 　炎の玉を生み出し、周辺の悪魔を回復する能力を持つ。
# ・ダムドルーク
# 　高い体力と高威力のビームを放ち、ポーン、ルーク、ビショップを生み出
# す力を持ち、キングと入れ替わるキャスリングを行うことが出来る。
# ・ダムドクィーン
# 　高い体力をもち、高威力の突進を行う。
# ・ダムドキング
# 　全ての駒を統括するキング。高威力のエネルギー爆発を起こす事ができる。
