
#　　属性がらみは原作を無視しています。ただし、字面、エフェクトから納得のいかない変更が
#　ある場合は、ミスの可能性があるのでどうぞ突っ込んでください。

#　人間：最大火力と成長性が売りでしょうか。武装の天井が一番高く、また成長させれば技能は
#　高くなりますし、高レベルで強力な精神コマンドを習得するキャラもそれなりにいます。半妖
#　も基本的にこれに準じます。

#　妖魔：遅成長が標準です。また光属性が弱点で、技の天井も低く、強力なスペシャルパワーもあま
#　り習得しません。技能も固定です。
#　　ただし、技能初期値とパラメータが高く、スペシャルパワーは最初から五つか六つ使えます。
#　　妖魔の君、及びそれに並ぶ格の妖魔は威圧＆魅惑＆不屈を標準装備しています。また、上級妖魔
#　は三つのうち二つ、下級妖魔は一つないし二つ習得しています。
#　　妖魔武具は、終盤をにらんである程度強力なモンスターを憑依させていることを想定しています。

#　メカ：ＳＰの成長速度が遅いですが、戦闘中ＳＰが回復します。また、技能も固定です。
#　　武装の天井は高めで耐久性に優れます。ただし、基本的に性格が機械で戦闘が長引くと不利に
#　なります。
#　　特殊効果を色々無効化するのですが、ジャマーとかも効かなくなってしまうので、その辺は
#　オミット。代わりに、武装の方に精属性を付けています。
#　　また、拡張では珍しい乗り換え対応ユニットにしてみました。換装は封印するのがめんどうだ
#　ってのが理由ですが。Ｔ２６０Ｇ以外はFix推奨です。

#　モンスター：技能は成長しませんが（というか、技能持ちが少ない）、精神コマンドは成長し
#　ます。技の天井は低いですが、燃費が良く、継戦能力に優れます。
