0, 0, 0, 0, 0
パンチはめ, 2200, 1, 1, -15, -, 80, -, AAAA, +20, 突

# コーディーを基準としています
# 攻撃力はハガー・Ｚ＝コーディー2300、Ｚ＝ガイ2200、ガイ2100になります。

# ファイナルファイト最強（凶？）の必殺技。一回あたれば最後、相手が逃れることはほぼ不可能
# また、与えるダメージもさることながら、プレイヤーの側にも強靭な精神力、忍耐力、技術力を必要とする。
# コーディーとガイで実行するのが有名だが、一応ハガーでもできる。
# バランス崩壊を招きかねないので武装に追加する時は慎重に。
# 特にコーディーにつける場合は「魂」を別のＳＰに変更してください。


# 気力制限は無し。キャラごとの威力の違いは原作ではたいして意味はありませんが、
# バランスを考慮して区別しました。命中率はやはり大技はあたりにくくするという原則から
# 低めに。原作でも数打ちゃあたる系でなく、一発あたらないと全弾くらわないタイプなので。
# ＥＮ消費が膨大なのは気力制限無しのハンデということと、データ製作者としてパンチはめに
# 頼った戦い方はなるべくしてほしくないという願いがこめられています。（もしかして開発スタッフの心境に近い？）
