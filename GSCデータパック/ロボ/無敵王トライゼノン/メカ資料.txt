ロボット編

トライゼノン
# 全長：35.4m　重量：318t(トライゼノン)
# 　　　22.2m(天号)　16.2m(冥号)　28.5m(海号)
# 北海道・夕張炭坑から掘り出された『遺産』。
# 地上における格闘戦を重視した機体である。
# 反面射撃武装が貧弱で、空中の敵相手には振り回されがち。
# ロケット推進システムが海号にしか搭載されていないため、
# 本来であれば宇宙での戦闘もままならない……はずなのだが、
# 神威章はアンカーのロケット推進システムをスラスター代わりにすることで
# 強引に宇宙での運用を行なっている。
# 分離形態の３機には正式名称として『天舞・冥穰・海皇』の名があるらしい。
# (14話・トライゼノンのモニター画面にこの３つの名がある)

トライゼノンＪＢ
# 最終回(22話)で、ガイアゼノンの両腕(ガイアコンドル・ガイアイーグル)を
# 両手に持ってゼノンエリシュオンに特攻した時のトライゼノン。
# イベントくさいのでデフォルトにおいては分離するようにしています。
# また、手を使う武器はほぼ撤廃しています。
# (スーパートライアタックだけは蹴りが使える、と言うことで残しました)

ＢＬＡＺＥトライゼノン
# コミック『トライゼノンＢＬＡＺＥ』にて登場した『真のトライゼノン』
# トライゼノンの全身に追加装甲を纏ったような格好となり、各種能力がアップしている。
# その昔、暴走したエリシュオンを封じたが、
# その力の強さゆえに当時のゼノパレス人に封印された形態……というのが設定である。
# なお、分離機能は排除されている。
# (というか、ＢＬＡＺＥだとトライゼノンも分離しない(実質)一人乗りだし(^_^;))

ガイアゼノン
# 全長：40.4m　重量：622t(ガイアゼノン)
# 　　　20.8m　　　　18.2t(ガイアイーグル・コンドル)
# 東京にあった『遺産』。
# 砲撃戦を想定した武装構成となっている。
# 設定では『継続的な』飛行はできないと明記されているが、
# 扱いはメディアごとにばらばら……(^_^;
# ＳＲＣでは(ユニット特性から)飛べない方がらしいかと思い、
# 飛行機能はオミットしました。
# (なお当初設定では、ＴＶのようなホバー移動は一切行なわず、
#  (イーグルとコンドルに付いている)キックモーターを使って
#  ジャンプ移動することになっていました)

アルマゼノン
# 全長：37.8m　重量：270t(ヒューマノイドフォーム時)
# 鹿児島より発見された『遺産』。
# 空戦を重視した機体である。
# その性能は樹雫の精神状態に大きく依存しており、
# コンディションが悪い時にはまっとうな攻撃・防御を行うことすらできない。
# 逆に雫が完全に覚醒した時、アルマゼノンは人型に変形することが可能となり、
# その能力は著しく上昇する(小説『アルマゼノン』より)

戦艦御来光＆東雲・暁・曙
# 全長：195.3m　重量：24000t(東雲)
# 地球に落ちし『希望の光』
# 300年前に起きたエストーリオの内戦の最中、劣勢になったオシリスの人間が、
# 地球人自らの手でゼノパレスの武力侵攻に対抗出来るようにするための手段として、
# ３体のゼノンズガーディアンとともに地球に送り込まれた。
# 本来の形は『御来光』であるが、地球にやってきた際３機に別れ、
# それぞれ北海道(東雲)、東京(暁)、鹿児島(曙)に落ちた。
# (分離して落下したのは、遺産の継承者たちがいきなり宇宙に飛び出し、
#  エストーリオに奇襲するのを防ぐための措置であると思われる)
# ３機がともに覚醒することで本来の姿、御来光に戻り、
# 単独で大気圏を突破するだけの出力をたたき出すことができる。
# なお、３機のうちの２機だけでも合体することが可能で、
# 東雲と暁の合体形態を『日の出』と呼称、他に東雲と曙の合体形態が確認されている。
# (御来光の合体構造を考えると、暁と曙でも合体は可能だと思われる)
# 主砲に相当する攻撃を行うと、エネルギーを一気に消費し、
# 暫く兵装出力が大幅に低下する弱点を持つ。
# (ただし、日の出は東雲単独時と出力が変わらない代わりに主砲発射に伴うタイムラグもない)
# また、御来光には高エネルギー攻撃を防ぐ防御障壁が存在している。
# (小説・ゼノパレス編から推測するに『グラヴィトン・ダイク』という名称らしい)

ゼノンガディオス
# 小説『ガイアゼノン』３巻登場。
# 『神の怨念』の別名を持つゼノンズガーディアン。
# タッカー＝ハインマンが自らの体にガーディアン操縦資格者の
# 遺伝子を無理矢理打ち込むことで操縦していた。
# 回避の概念を持たず、圧倒的な火力と防御力で敵をねじ伏せるコンセプトの元作られた。
# しかし、それを実現するための手段である大出力のビーム砲と
# バリア『グラヴィトン・ダイク・ゲテム』の存在により、継戦能力が極めて低くなってしまうが、
# タッカーはこの弱点を、機体と要塞のメイン動力炉を直結させる事で補った。
