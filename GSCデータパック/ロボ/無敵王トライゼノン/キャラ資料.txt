# トライゼノン＆東雲搭乗員

神威章
# 本作の主人公で、トライゼノンメインパイロット／天号パイロット。

# お宝＝トライゼノンを発見し、そのパイロットとなる。
# 典型的な熱血バカ。その一言に尽きる(爆)
# (ＣＶ：福山　潤)

# トライゼノンを兄とともに発見した際、冥号のパイロットとして登録される。
# バカな父親と兄を反面教師にしたか、極めて計算高く、したたか。
# 趣味は株取引というから末恐ろしいというか……(^_^;
# (ＣＶ：真田アサミ)

雨竜華菜

# トライゼノン誕生の際、たまたまその場に居合わせ、
# 海号のパイロットとして登録される。
# はちゃめちゃな東雲乗員に振り回されまくる不幸な少女。
# そんな境遇を常に嘆いているが、時たまキレて暴走するのはご愛敬(笑)
# (ＣＶ：林原めぐみ)

神威ゴン
# 神威家の飼犬(ゆえに識別の意をこめて名字を付けさせていただきました)

# 史上初(?)の犬の艦長である(核爆)
# なお、合体システムの関係上、日の出及び御来光の艦長も兼ねる(^_^;
# (ＣＶ：渋谷　茂)

神威権太郎
# 神威章・哀兄弟の父親。
# 章に輪をかけたバカ(爆)であり、章に先祖伝来のお宝を取られたのが悔しくて、
# 東雲や暁・曙をわが物にするべく暴れ回る。
# 元・夕張炭坑の坑夫であり、つるはし片手に敵を破壊しまくる、まさに非常識人(爆)

雨竜冴

# 自衛隊の三等陸曹であり、ヘリ部隊のエースガンナーとして活躍していたが、
# トライゼノン搭乗者の知り合いと言う理由で、監視役として東雲に出向させられる。

デュラン＝クレネード
# ゼノパレス・地球降下軍司令(上級大将)／ゼノンノーヴァパイロット。
# 両親ともにマーズ軍治安部隊・ラヴェーダのパイロットだったが、
# オシリスの暴徒に殺されたと両親の部下であったレオン＝アーヴァインに教えられ、
# 敵を捜すべく自らもラヴェーダの一員となる。
# そのとき偶然フラウディアと出会った事でマーズ軍の虐殺行為を知り、
# さらには両親が虐殺行為からオシリスを救おうとしたためにレオンに殺された事実を知らされる。
# そして彼は、両親の敵を討つため、両親がオシリスに遺したゼノンノーヴァを駆り、
# フラウディアと共にオシリスのため戦う事になる。
# (以上、小説ゼノパレス編設定)
# オシリスとマーズの和解に成功し、ゼノパレス軍のまとめ役となった彼は、
# 地球占領に際し最も大きな障害である『遺産』(=地球に渡されたゼノンズガーディアン)
# の奪取もしくは破壊を実行するため、ゼノパレス地球降下軍の司令として地球に赴く。
# しかし、地球の自然をできるだけ残すため、無益な破壊活動はするべきでないと主張する。
# 地球人類との交戦に心を痛めながらも、病の床にある妹・セシルと
# 種として絶滅の危機に瀕するエストーリオの民を救うべく戦う。
# 極めてまじめで、それゆえに御来光艦長がゴンだと知った時一瞬精神崩壊を起こす(爆)
# (ＣＶ：渋谷　茂)

# デュランの妹。
# 生まれながら一部の遺伝子が欠損しており、病院暮らしを余儀なくされる。
# デュランが地球にこだわるのはセシルを本物の自然がある地球に住まわせるため。
# なお、甲斐童夢と遺伝的に同一のものを持っているとされ、
# (正確には童夢の遺伝子と遺伝子欠損発生前のセシルの遺伝子が同一であると推測される)
# 時たま童夢とテレパシーのようなもので心がつながることがある。
# (ＣＶ：林原めぐみ)

フラウディア＝アドリス
# ずば抜けたカリスマ性によって人々の信頼を集めるエストーリオの最高指導者。
# その正体は、1000年の時を経て遺伝子状態から復元された、

# 第二次黎明戦争(小説ＺＰ編)後、ロルムス等の手により洗脳され、傀儡と化す。

# また、彼女の血筋を考えると、ゼノンズガーディアンを操縦できてもおかしくないので、
# 機体クラスに『ゼノンズガーディアン』を入れました。

# トライゼノンＺＰ編

クラーク＝ルイン
# オシリス・カレッサシティの指導者。
# フラウディアにアルシオーネの面影を見た彼は、
# 『オシリスでマーズ軍が虐殺をしている』という情報を流して彼女を呼び寄せ、

# フラウディアがそれを了承した後はレジスタンスの指導者として活躍。
# どちらかと言えば頭脳労働が専門だが、メタアーマードの操縦も十分こなす。

イシュター＝ワング
# オシリス・レイナルシティのリーダー。
# 父にしてゼノンバルバロッサのパイロットであるパブロ＝ワングから
# 幼きころより『騎士道』をたたき込まれた女丈夫。
# それゆえにどちらかと言えば軟弱な類に入るクラークをあまり好ましく思っていない。
# が、後にはクラークとともにデュランを戦闘面で支えるようになる。
# 理論上、彼女もゼノンズガーディアンを操縦できる可能性があるため、
# 機体クラスとして『ゼノンズガーディアン』を組み込んでおきました。

パブロ＝ワング
# ゼノンバルバロッサパイロット。
# レイナルシティの指導者だったが、『条件を飲まなければシティの人間を抹殺する』という
# レオンの脅しに似た取り引きに応じ、マーズの戦士『ワイズ』として戦うことになる。
# デュランとの戦いに破れた後、部下をフラウディアに託して自決しようとするが、
# フラウディアの慈愛に満ちた言葉に心を打たれて彼女に忠誠を誓い、
# 愛機・バルバロッサを失った後も、デュランと並ぶオシリスのエースとして活躍する。
# なお、ＴＶ本編の『パブロ』との関係ですが……おそらく別人でしょう。
# (というか同一人物と認めたくないです、ＴＶでのあの情けなさ見ると(^_^;))

レオン＝アーヴァイン
# ゼノンジェダインパイロット。
# ラヴェーダの一員として、デュランの両親らとともに戦い、
# デュランがラヴェーダに所属した後も彼の良き兄貴分として振る舞っていた。
# その正体はオシリス虐殺部隊『深紅の魔人(クリムゾン・ジーニアス)』の隊長であり、
# オシリスの力になろうとしたデュランの両親を殺した張本人。
# なお彼は、遺伝子欠落の影響で余命幾許もない体と化しており、
# その事実が『ゼノンズガーディアンの融合』という暴挙を導く。

バルザック＝アドリス
# マーズの長にして、ゼノンアルティメオスパイロット。
# エストーリオの指導者としてマーズの民から多くの支持を得る一方で、
# レオンを通じてオシリスの虐殺を実行する。
# さらにはオシリスに対する切り札とするべく、遺伝子バンクから
# アルシオーネの遺伝子を見つけ出して復元、娘として育てるが、
# その娘・フラウディアがオシリスの指導者となってからは全面対決状態となり、
# アルティメオスを駆ってデュランとの一騎討ちに挑む。
# このような行動に出たのは、エストーリオの持つさまざまなシステムが老朽化し、
# もはや全ての民の生活を支えきれないと判断した事に起因する。
# 一騎討ちの最中、この現状をオシリスに知られるが、
# それを知ってなお、あくまでも平和的な道を行くことを願う
# フラウディアの決意に心打たれてオシリスに降伏、和解の道を探ろうとするが、
# 直後にやってきたレオンによって抹殺される。
