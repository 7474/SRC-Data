# 多国籍企業ＧＡＩＬの私設軍について
# ＧＡＩＬ私設軍の装備は、その世界の「一般的兵器」です。
# 作中においては戦車・戦闘ヘリなどを使用していました。
# しかしＳＲＣにおいてはＭＳなどを使用するのが「らしい」と思われます。
# 企業繋がりで、ネルガルと提携させてエステバリスを出すのもいいでしょう。
# ただの戦車・戦闘ヘリはわざわざデータ化しても面白くないのでしません(笑)


# ガーディアンのアイテム装備数が多いのは

# でしょう。
