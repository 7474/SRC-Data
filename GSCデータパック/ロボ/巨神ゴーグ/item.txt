# 70mm砲は、戦車の主砲を改造してゴーグでも使えるようにしたもの
# 末尾の「！」はわざとです。
# パルスランチャーはラブルガーディアンから奪ったもの(本来はガーディアンの標準装備)
# このように、ゴーグはその場で手に入れた武器を使用します
# 作中においてはゴーグが戦った相手は
# ・多国籍企業ＧＡＩＬの私設軍
# ・マノン配下のガーディアン
# だけであったため、上記の武装でした。
# しかし、ＳＲＣにおいては「それ以外の敵」がほとんどでしょう。
# ですので
# ・バッシュを倒してバスターランチャーを奪う
# ・仲間のダグラムから、ターボザックの予備を回してもらう
# ・Ｎ２爆雷を手に持ち、それで殴る(爆発は装甲で耐える(笑))
# といった、各シナリオでのアイテムを駆使した強化対応をお願いいたします。
# できましたら、最初からゴーグ用に造られたというより「他のロボットのための武装を
# 有り合わせ的に装備」とした方が「らしい」かと思われます。
