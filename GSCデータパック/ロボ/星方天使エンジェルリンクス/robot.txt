#○エンジェルリンクス
#天才造船官ヒラガ＝ダインコート卿の設計によるアブソルート級宇宙戦艦を
#ベースに、ＰＫ(対海賊装備)を施した改良型。アブソルート級はＴＳ軍縮条約で
#全て建造キャンセルされているので、既知宇宙で唯一のアブソルート級と言えます。
#超高速の移動力、比類無き高火力、重装甲を誇ります。
#なお、小説版での名称は《エンジェル》。
#アニメでは、この形態のことを「フォーマル・ドレス」と呼んでいました。
#
#○武装解説
#・激光砲(ブラスター)
#　中国語で激光はレーザーを示す。文字どおりレーザーを放つ。
#・火箭連弾(ミサイルスリンガー)
#　核弾頭クラスのミサイルを連発し、広範囲にダメージを与える。
#・光槍砲(ビーム)
#　ビーム砲のこと。
#・天環砲(リンクスキャノン)
#　エンジェルリンクスの強さを決定づける兵器の一つ。
#　白く光るリングが当たった後に大きく現れる(小説では砲口にリングが現れる)。
#　出力は30～80％で放っていたが、その出力差を示す為にマップ兵器版をチャージとした。
#・神託砲(オラクルキャノン)
#　空間のエーテル流を乱し、効果範囲すべての機体の推進力を0にして
#　狂わせる兵器。初陣ではこれを放って民間機まで巻き込んでしまう。
#　苦情305件。
#
#○パイロット数に関すること。
#ブリッジには総勢10人(美鳳、鴻星、ドゥーズ、ヴァレリア、アンナ、アイハラ、
#ニコラ、セルジェ、ドック、ボーイ)いますが、実質パイロットとして
#定義するのは美鳳、鴻星、ヴァレリア、ドゥーズのみとし、ほかは
#ノンパイロットとします。ダイアログメッセージで上記４人のメインメンバー以外を
#喋らせたくなときはLeaveコマンドを使って下さい。
#
#○短距離サブイーサについて
#この世界においては、サブイーサ（一種のワープ航法）は短距離では行えません。
#が、エンジェルリンクス号は美鳳を生み出した遺跡の技術を解析することにより、
#短距離でのサブイーサを可能としています。ただし、これは小説版のみで、
#しかも美鳳の正体が明かされた後、美鳳を生み出した遺跡を発見してからでないと
#使うことができませんので、それまでは封印しておいてください。
