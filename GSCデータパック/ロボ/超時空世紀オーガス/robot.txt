# エマーン陣営 ###########################################################################
# エマーンのメカは「浮かぶ腕(足のないＶＦ－１ガウォークって言えば解り易いかな？)」と言えるものが
# 基本です｡武装もミサイルや機関砲が基本でビーム兵器はほとんどありません。
# (エマーンは商業国家なので一度売った後でも弾薬の需要があるミサイル等をメインに開発しています)
# そして慣性制御システムで自在に空を飛び、感応システムで素早い動きが可能というスグレモノです。
# (ブロンコⅡと大尉もここに分類しました)
#
# ●オーガス -----------------------------------------------------------------------------

# (この時、桂の「腕が無いと木が掴めない！」というリクエストで腕も付きました(笑))
# バルキリーの上をいく４形態への変形機構を持つ。オーガスⅡは量産タイプです。
#
# 「オーガス(前期型)」はシナリオ序盤用、「オーガス」はシナリオ中盤からの使用を想定して
# バランスをとってあります。
