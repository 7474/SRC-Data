
# ビームサーベル系とビームライフル系の武装をそれぞれ１つずつしか持っていない
# クァイア・シリーズには、救済策としてムジカの台詞「音速の刃」からの捏造技、
# 『ソニックブレード』を追加してあります。
# 基本的に、作中でビームスライサーの威力が 2400 だった機体に付加してあります。


# 格闘武器主体の機体は移動力を高めにしてあります。


# クァイア、及びカリオン(そしておそらく旧ウルス兵器も)に使用されている
# 操縦機構『脳波シンクロシステム』と、カリオン及び旧ウルス兵器に使用
# されている『サイキックシステム』は、特にデータ化していません。
# サイコスティンガーやサイキックブレードには武器属性サが付いていますが(^^;


# クァイア・シリーズ専用機、クァイア・エボリューション、クァイア・アームド、
# カリオン・インフィニート、旧ウルス兵器以外の S-M-A-P は基本的に全て量産機
# なので、シナリオでのザコとしての使用が可能です。


# 作中に隠しキャラとして登場した超機大戦ＳＲＸでおなじみのリュウセイ＝ダテ
# ですが、彼は S-M-A-P に乗り換えが可能でした。
# 恐らく S-M-A-P の操縦系統がＰＴと同じ脳波コントロールを使用しているからだと
# 思われます。
# したがって、他のバンプレストオリジナルキャラでも S-M-A-P に登場可能なように、
# 機体クラスをＰＴに変更してあります。
# (パンプレストオリジナルキャラで、ある程度の互換性を持たせるという意味合いも
# 強いですが)
