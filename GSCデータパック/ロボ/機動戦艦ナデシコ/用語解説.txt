
# ナデシコ・ユニット関連基本解説

# ●エステバリス

# ナデシコ世界で味方側（ネルガル重工、地球連合軍）が使用する人型ロボット
# です。全高８ｍ前後、最大の特徴は頭部コクピットユニット（アサルトピット）を
# 軸として、様々な用途・状況に応じた機体フレームを自由に交換できることです。
# アサルトピットはパイロットごとに形状が違うので、改訂に際しては各パイロット
# 個人のアサルトピットを「機体」として作成し、各種基本フレームはアイテムとして
# 「フレームをアイテムとして装着」するような形で仕上げています。
 
# エステバリスはエネルギーリンクと呼ばれる、母艦から重力波によるエネルギー
# 供給を受けており、母艦から離れない限りはエネルギーが回復する能力があり
# ますが、データ上は反映されていません。




# ●ディストーションフィールド
#
# 重力制御を用いて、時空を歪曲させバリアとするものです。
# 主として光学兵器への対抗手段ですが、原理上実弾にも効果があります。
# 「バリア」としてデータ化してあります。




# ●ディストーションナックル

# エステバリスは劇中、ディストーションフィールドを使用した「決め技」を
# よく用いました。つまり拳やナイフの先に纏って攻撃したり、全身にまとって
# 敵中に突撃したりです。これらは、すべて「ディストーションナックル」に
# まとめてデータ化してあります。
# 通常のフレームでこれを使用するには、「熱血バワー」が要ります。




# ●グラビティブラスト

# 重力制御技術を用いた、一種の重力波レーザーです。
# 新型艦船の主砲級という扱いで、この世界では非常に強力な兵器に属します。
# 目で見ると「陽炎をともなう黒い光条の束」に見えます
# 陽炎をともなう、というのは発射口の周辺空間が歪んでいるからです。



# ●ボソンジャンプ

# 作品のメインキーともなっている、古代火星人の遺産である遠隔転送技術です。
# 厳密には「テレポート」ではなく、時間をコントロールすることにより移動を
# 瞬時に行っている、というものです。
# そのため、制御を誤ると「２週間前の火星」に「跳んで」しまうこともあります。

# 遠距離のものはイベント扱い、機動兵器の行う短距離のものはせいぜい一度に
# 数百ｍのものであるため（連発は可能ですが）、移動力を増強するというより
# これを利用した回避効果がふさわしいと考え、「超回避」としてデータ化して
# あります。生体のボソンジャンプは特殊な遺伝子操作を行った生物（木星連合
# の優人部隊や、火星の環境ナノマシンに遺伝子を改良された火星生まれの人
# 間）しか行えません。
# 劇場版では、どの程度ボソンジャンプを自在に制御できるか＝火星の遺跡との
# シンクロニティによってレベル分けがなされています。
# アキト、ユリカ、イネスなどはＡ級ジャンパーに分類され、戦艦を丸ごと
# ボソンジャンプさせることすらできます。
# ルリはＢ級ジャンパーの資格を持っています。



# ●バッタなどの無人兵器

# 敵・木星連合（木星蜥蜴は地球側からの通称）は、虫型などを模した無人
# 兵器戦力を主力としています。無人兵器はおおむね虫などを模しており、
# 人間大から戦艦級まで様々です。味方兵器のコンピュータシステムを乗っ取る
# こともあります。パイロットとしては、「木星製ＡＩ」を推奨。



# ●木連系機動兵器

# 敵・木星連合の切り札は、ボソンジャンプが可能な遺伝子操作を施された
# エリートによる有人兵器部隊、優人部隊です。使用する兵器は、無人艦艇とは
# フォルムの異なる艦艇と、ジンシリーズなどの人型兵器からなっています。
# ジンシリーズは木星連合が聖典とあがめるロボットアニメ、「ゲキ・ガンガー」
# の登場メカにそっくりの形状をしており（ナデシコ側からはゲキガンタイプと
# 通称されています）、グラビティブラスト、ディストーションフィールド、
# 単体での短距離ボソンジャンプによる奇襲攻撃能力を備えた強力な兵器です。
# ジンはボスクラスの敵として使用する場合は「ボスランク」を使用して下さい。
# 後半戦であれば、ボスランクを使用せずに味方ユニットとして使用することも
# 可能です。


# データの乗り換え種別

# ＩＦＳ               操縦にはナノマシンを使用したＩＦＳが必要。
# 木連系機動兵器       木連の技術で開発された物。
# ＩＦＳ非対応         ＩＦＳを使用しない機体等。。
# 機動戦艦戦艦         ナデシコ世界での戦艦。
