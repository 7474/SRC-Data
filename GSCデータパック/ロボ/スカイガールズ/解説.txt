## ソニックダイバー隊 ########################################################

## 桜野音羽　　ＣＶ：川澄綾子
#
# 　第二期ソニックダイバー隊。搭乗機は零神。階級は曹長。
# 　祖父の営む桜野無敵流道場の娘で、跡継ぎを目指しつつ海女で生計を立てている。
# 肝心の剣術はそれなりの腕前を持つが、才能は未知数。
# 　天真爛漫で男女の分け隔てなく接するタイプ。他人を思いやる気持ちが強いのだが、
# 直情型の性格のため親切がおせっかいに発展することも多い。また、弟の影響で大空
# に憧れを持っている。
# 　ソニックダイバーの操縦は学課が今ひとつなものの、ＭＯＬＰは非常に高く同調率
# は異常ともいえるほど。しかし、精神状態に大きく左右され、急上昇することもあれ
# ば下降することもあり、安定していない。特にワームの人間型メッセンジャーとして
# 弟の桜野優希が現れてからしばらくの間はＭＯＬＰが上がらず、ソニックダイバーを
# 動かせなくなった。
# 　ＴＶでは出身が瀬戸内海の孤島となっているが、小説では沖縄となっている。


## 園宮可憐　　ＣＶ：後藤沙緒里 
#
# 　第二期ソニックダイバー隊。搭乗機は風神。階級は曹長。
# 　いわゆる天才であり、飛び級を重ね１６歳にして仙台高等大学院にて博士号を目指
# すほど。メカニックとプログラムの権威であり、戦闘時のプログラムテクニックは神
# 懸っているといえる。そのＩＱは200(ＴＶ版。小説、ＯＶＡでは170)ほどもあり、あ
# らゆる事象を数値化することが出来る。その特技をもって、隠れ巨乳である藤枝七恵
# のバストのサイズを見破った。
# 　おっとりとした性格で、人と話がかみ合わないことも多い。ただしそれは単純に天
# 然だと言うのではなく、頭の回転が速すぎるため会話がズレてしまうためである。ま
# た、世間知らずで人見知りが激しく、特に兄を除く男性とはまともに顔を合わせるこ
# とも出来ず、会話は友人を介して行っていた。
# 　ソニックダイバーの操縦で特に優れた面はないが、総合的データ解析能力は高く、
# 戦闘時にはその分析力をもって即座に対処法を立案し、ソニックダイバー隊全体の行
# 動をサポートしている。


## 一条瑛花　　ＣＶ：伊藤静
#
# 　第二期ソニックダイバー隊リーダー。搭乗機は雷神。階級は上級曹長。
# 　父に憧れパイロットを志すも反対され、家を飛び出して軍に志願した（ＴＶ版。小
# 説版では父は元法務将校であり、関係もさほど悪くはない）。そこで努力し、若くし
# て春日基地（ＴＶ版。小説では横浜基地）のエースパイロットにまで駆け上った。し
# かし、そこで試作型ソニックダイバーと戦闘を行い、惨敗を喫したことでソニックダ
# イバーの将来性を知り、ソニックダイバー隊に志願した。
# 　才色兼備でプライドが高く負けず嫌い。才能にも恵まれているがそれに甘んじるこ
# となく努力を怠らない。それは将来が嘱望されたエースパイロットと言う立場を蹴っ
# て勝利を求めソニックダイバー隊に志願したことからも伺える。また、堅くきつい性
# 格のようにも思えるが、物言いがきつく自分に厳しいだけで、本質的には優しい人物
# といえる。
# 　甘いものに目がなく、怖い話が苦手。
# 　ソニックダイバー隊唯一の軍属であり、さらに努力家と言うこともあって学課とシ
# ミュレーター訓練で良好な成績を残し、隊で一番に実機訓練を行った。しかし、スカ
# ウトではなく志願であるため、音羽や可憐と違いＤＮＡ適正が低く、そのことで最初
# は苦労することになる。
# 　しかし、じきに安定して操縦できるようになると、元戦闘機パイロットの高い判断
# 力と持久力により隊を引っ張る頼れるリーダーへとなっていった。


## エリーゼ＝フォン＝ディートリッヒ　　ＣＶ：辻あゆみ
#
# 　西ヨーロッパ方面軍西ヨーロッパ基地第９特殊空挺団所属。しかし、基地はワーム
# の襲撃により壊滅してしまい、支援要員として攻龍に派遣された。搭乗機はバッハシ
# ュテルツェＶ－１。
# 　父親と母親は軍属であり、両親亡き後も基地で育ち基地の人員と家族のように生活
# してきたが、ワームに襲撃され基地は壊滅する。その時エリーゼのみ脱出させられ、
# 基地はエリーゼの前でワームを道連れに先の大戦で使われた大量破壊兵器により自爆
# した。
# 　根は素直だが、素直すぎて子供っぽい。勝気で思ったことはすぐ口に出るため、瑛
# 花との意見の対立がたびたび起こる。しかし、その子供っぽいところが孫がいるよう
# な世代の隊員にはかわいいらしく、実の孫のようにかわいがられている。また、攻龍
# に送られた理由からワームを極度に憎んでいる。
# 　ソニックダイバーの操縦は乱暴ながら腕は確かで、瑛花に引けをとらない。また、
# 最初はチームに馴染むことが出来なくて独断専行も多かったが、仲間として認め打ち
# 解けたことでスムーズに連携を行えるようになった。


## アイーシャ＝クリシュナム　　ＣＶ：片岡あづさ
#
# 　謎の多いインド系と思われる少女。外見年齢は17歳ほどだが、実年齢は31歳。
# 　クリシュナム博士の実子であり、不治の病を患っていた。それを治療するために博
# 士は生体ナノ技術の臨床実験をアイーシャを材料として行っていたが、そのことがば
# れスキャンダルになり行方をくらます。その後、インドの山奥に人工冬眠させられて
# いたが、その１８年後に発見。目覚めた後は、ソニックダイバーの開発に深くかかわ
# った。
# 　非常に無口な性格で、感情の起伏自体に乏しい。
# 　ソニックダイバーの操縦技術は本来パイロットでないため非常に低い。しかし、完
# 全な人機一体が可能で、ＭＯＬＰ値は常に１００％を超える。その反面、操縦中の記
# 憶は失われてしまうため、過去に瑛花との模擬戦を行った事実を忘れてしまっている
# (ＴＶ版。小説版では番匠真里亜が瑛花と模擬戦らしきものを行った)。


## 一樹楓
#
# 　第一期ソニックダイバー隊テストパイロット。搭乗機は試作型零神。
# 　スピードに優れる零神を乗りこなし、遷音速まで14秒と言う驚異的な記録を叩き出
# すも、その試験終了後に帰ってきたのは零神のみだった。
# 　小説版のみ登場。


## 水崎香南
#
# 　第一期ソニックダイバー隊テストパイロット。
# 　非常に優しい性格で、いつも険悪な冬后、緋月の仲をなんとかしようとしていた。
# 　ソニックダイバーの操縦は楓や真里亜にスピードで劣るものの、的確な分析力やテ
# クニックでそれを補っていた。
# 　楓の死亡、真里亜の事故負傷後に軍を後にした。
# 　小説版のみ登場。


## 番匠真里亜
#
# 　第一期ソニックダイバー隊テストパイロット。
# 　一人称が「おれ」で、粗雑な印象を受ける性格をしている。また、冬后のことを
# 「ソーヤ兄貴」と慕い、その行動を真似ることがたびたびあった。
# 　ソニックダイバーの操縦においてスピードは楓にかなわず、テクニックでは香南に
# 一歩及ばないが、その代わりに攻撃力は一番優れていた。また、適正の問題なのか初
# 期の生体同期システムによる不具合なのか、ＭＯＬＰが安定しなかった。
# 　事故による負傷後、軍の基地内で療養を続けていたが、その基地がワームによる襲
# 撃を受ける。それを知った緋月がソニックダイバーの情報がワームの手に渡ることを
# 恐れ破棄しようとするが、真里亜はそれを阻止するため事故による後遺症で不自由な
# ままの体でソニックダイバーに搭乗。実験段階だったＭＭ－ＢＷミサイル二基を抱え
# ワームに特攻を行った。
# 　小説版のみ登場。

## 追浜基地／攻龍クルー ######################################################

## 門脇曹一郎　　ＣＶ：小川真司
#
# 　特務艦「攻龍」艦長。階級は中将。
# 　非常に無口でありほとんど発言する所を見られないが、ジョークに理解を示したり、
# 食糧危機の折にソニックダイバーを漁に使うことを許可したり、艦内の写真を撮るた
# くみに対して自分は撮らなくていいのかと聞くなど、堅物と言うわけでもない。
# 　大胆さと用心深さを併せ持った「門脇流」戦術をもった優れた艦長である。また、
# 出身は見張り員であり、ＥＣＭでほとんど使い物にならなくなったレーダーより先に
# ワームを夜間に目視で発見した。


## 嶋秋嵩　　ＣＶ：斧アツシ(ＴＶ)／青山穣(ＯＶＡ)
#
# 　特務艦「攻龍」副艦長。階級は少将。実質的な戦闘指揮は彼が取っている。
# 　ソニックダイバーを運用するための艦の副艦長でありながら、兵器としてのソニッ
# クダイバーの存在には懐疑的。そのため、ソニックダイバー隊やその上官である冬后
# の意見をないがしろにするが、艦長がその意見を取り入れ指示することも多く、その
# 場合は逆らうことが出来ない。


## 冬后蒼哉　　ＣＶ：藤原啓治
#
# 　ソニックダイバー隊の指揮官。正式な肩書きは「統合人類軍 極東方面大隊 第十三
# 航空団所属 試作特機実験部隊指揮官」という長いもの。もとは大尉だったが、攻龍
# に配属が変わる時に昇進し中佐となった。
# 　過去には黒騎士と呼ばれ、いかなる激戦からも生還するエースパイロットだったが、
# ワームとの戦いで両腕を負傷。引退せざるを得なくなった。その因縁の相手とは、後
# にソニックダイバー隊とビックバイパー隊を苦しめることにもなったカニの姿によく
# 似たワームである。
# 　その後、飛行隊の教官などの就任要請を固辞しながらだらだらと過ごしていたもの
# の、半ば騙される形でソニックダイバー隊の指揮官とされる。
# 　趣味は釣り。基地が海に近いこともあり、ほぼ毎日興じている。


## 緋月玲　　ＣＶ：間島淳司
#
# 　技術開発本部所属だが、現在では出向と言う形で冬后の副官となっている。階級は
# 少尉。
# 　民間からソニックダイバーのパイロット候補者を集めた。また、副官でありながら
# 上層部とのつながりも強く、ソニックダイバーに関する多くの情報を持っているため
# 上官であるはずの冬后に対し強い発言力があるように見受けられる。ソニックダイバ
# ーの開発者である周王博士と密接に連絡を取り合っている。
# 　性格は非常に冷静で感情を表に出さない。また、ソニックダイバーの情報を集める
# ため冷徹な判断を下すことも厭わない。それは、ソニックダイバー乗りの死亡もデー
# タとして有用と言い切るところなどからも伺える。
# 　攻龍の完成により部隊は攻龍へと移るも、緋月は地上勤務として残る。しかし、そ
# の後も重要な局面には攻龍へと顔を出していた。


## 藤枝七恵　　ＣＶ：堀江由衣
#
# 　情報処理兼航空補佐。猛勉強の末に軍の特別徴収試験を受け見事に合格。中学生に
# して軍属になる。その猛勉強の際、極度の近視となり眼鏡がなければ何も見えなくな
# ってしまった。
# 　まじめで几帳面、さらに六人兄弟の長女と言うこともあってかしっかり者で落ち着
# きもあり、世話好きのおせっかい焼き。さらに学年で常に一番という、性格の良さと
# 優秀さを兼ね備える才女だが、自分を上回る能力を持つソニックダイバー隊の三人の
# 存在に少なからずショックを受けたりもしている。
# 　本人が気にしておりバッグや書類でいつも隠しているが、実はかなりの巨乳。最初
# はそれを隠し通し音羽から「貧乳仲間」扱いされていたが後に風呂場で「裏切り者」
# 呼ばわりされることになる。
# 　動物好きで、軍規に反するものの秘密でハムスターを飼っている。


## 速水たくみ　　ＣＶ：白石涼子
#
# 　通信士兼厨房補佐。媒体による設定のズレが登場人物の中で一番大きい。
# 　ＴＶ版では、食堂を営む実家の家計を助けるため軍に志願するが、いつかは帰って
# 店を継ぐために本来の通信士の仕事のほかに厨房補佐を志願して兼任している。
# 　小説版では、パイロットに憧れていて、自らもパイロットになるのが夢だったが、
# 才能のなさなどから断念。パイロットの補佐をする道を志している。
# 　ＯＶＡ版では、将来は自身の設計した飛行機で空を飛びたいと夢見ており、暇な時
# 間を見つけては設計図を描いている。
# 　共通する趣味はカメラ。古い町並みや攻龍の艦内の撮影を行っている。
# 　温厚な性格で、若さと経験不足から来る優柔不断さを除けば仕事は的確。


## 大戸代治　　ＣＶ：田中完
#
# 　整備兵班長兼追浜基地、及び攻龍メカニックの責任者。後にバッハシュテルツェＶ
# －１専任のメカニックも担当する。これはエリーゼが急に来日したため手が空いてい
# るメカニックがいなかったための特別な措置である。
# 　口は悪いが腕は確かで、人を見る目と人望に優れた優秀なメカニック。「いかに機
# 体に乗るパイロットを安心して飛ばせられるか」という独自の哲学を持っている。
# 　上記のような人柄から、若い部下からは父親のように慕われている。また、エリー
# ゼを孫のようにかわいがる一人でもある。


## 橘僚平（橘遼平）　　ＣＶ：谷山紀章
#
# 　零神専属メカニック。腕は天才的なのだが、その軽い性格からとてもそうは思われ
# ない。
# 　夢はバイクをアクセル全開で走らせること。しかし、今の絶対的石油・鉄鋼資源の
# 不足から石油は高騰しており、必死にお金をためても町一周するのが関の山なのが現
# 状である。
# 　ＴＶシリーズ以降は橘"僚"平だが、ＯＶＡ・アートワークス・小説では橘"遼"平と
# いう名前で記載されている。


## 御子神嵐子　　ＣＶ：比嘉久美子
#
# 　雷神専属メカニック。晴子の双子の姉。
# 　明るい性格で、誰からも好かれる。また、似たような性格の音羽と気が合う。
# 　自己紹介いわく「力の嵐子」。
# 　大雑把だが、整備の腕前は確か。


## 御子神晴子　　ＣＶ：石川桃子
#
# 　風神専属メカニック。晴子の双子の妹。
# 　ドライで淡白な性格だが、口数が少ないだけで笑顔も時々のぞかせる。
# 　嵐子の紹介いわく「技の晴子」。
# 　整備はメカニック一正確誠実。


## 源さん　　ＣＶ：武虎(ＴＶ)／河合義雄(ＯＶＡ)
#
# 　追浜基地、及び攻龍の料理長。
# 　たくみにとって料理の師匠であり厳しい親方でもある。しかし、死んでしまった孫
# 娘によく似たエリーゼに対しては特別にデザートを作るなど、非常に優しく接する。


## 安芸夕子　　ＣＶ：生天目仁美
#
# 　ソニックダイバー隊専属の医療スタッフ兼特務艦攻龍の船医。
# 　肉体的な怪我や病気にとどまらず、メンタル面でもクルーのサポートを行う。その
# ため、攻龍クルーで彼女に逆らえるものはほとんどいない。
# 　既婚者であり、夫は先の大戦で死亡。一人息子はいるが、実家に預けているので一
# 緒にいられないことを気にしている。


## 美空秋人
#
# 　ゲーム版オリジナルキャラクターであり、プレイヤーキャラクター。
# 　人類統合軍所属。25歳で、階級は少尉。15歳にして先のワーム大戦に後方ではある
# が参加している。そのとき配属していたのは門脇と嶋がいた部隊。
# 　指揮官補佐と言う名の雑用係として追浜基地に配属されるが、実際に作戦の指揮を
# 取ったり様々な部署の隊員と親交を深めながら認められていき、最終的には冬后の副
# 官に任命され、戦闘指揮を任されるほどの信用を得ることになった。
# 　本当はパイロットになりたかったものの、心臓に持病があり、日常生活はともかく
# 戦闘機の機動についていけるほどの心肺能力は得られないため、パイロットへの道は
# 断念せざるを得なかった。
# 　心臓の治療にナノシステム治療プログラムを行っており、そのことが後に大きな問
# 題を起こすことになる。

## ビックバイパー隊 ##########################################################

## 飛崎陣　　ＣＶ：諏訪部順一
#
# 　ビックバイパー隊。搭乗機はビックバイパー。階級は大佐。
# 　統合人類軍のベテランパイロットであり、先の大戦では冬后とエースの座を競い合
# うライバルでもあった。現在では空母天武に正式配備されたビックバイパー隊のリー
# ダーを勤めている。

## ソニックダイバー ############################################################
#
# 　日本海軍、技術開発本部が対ワーム用に（表向きには平和利用を目的として）開発を
# 行った新機軸の航空機動兵器。正式名称は『飛行外骨格　ソニックダイバー』。
# 　小型融合炉を搭載し、人型という特性からＶＴＯＬが可能。基本的な形態は高速巡航
# 時のＧ（グライダー）モードと、戦闘時のＡ（アーマード）モードの二種類だが、何ら
# かの理由でパイロットが操作不能になったときのために自動で行動するＡＳ（オートマ
# チックソルジャー）モードという形態も用意されている。
# 　パイロットスーツ（モーションスリット）などの細部にわたるまで軽量化を行った結
# 果、マッハ２を超える飛行速度や、圧倒的な機動性を得ることになった。しかし、軽量
# 化のため風防もなくパイロットは基本的にむき出しであり、僅かな被弾でも命取りにな
# りうる。また、僅かな重量差により挙動に影響が出ないようにするため、パイロットの
# 体重は５０キロ以下であることが求められる。
# 　むき出しのパイロットの保護はナノスキンジェルによって行われているため、超音速
# で機動してもパイロット自体に影響はない。しかし、ナノスキンジェル自体はホメロス
# 効果で21分32秒しか持たず、ナノマシンを塗布する「スプレッドブース」と呼ばれる装
# 置に適正な電力が供給されていないとさらに時間は短くなる。また、ナノスキンジェル
# は機体との連動性能に影響させないため１．５ミリまでの厚さしかなく、物理的防御力
# は９ミリ弾を止めることが出来る程度でしかない。
# 　生体同期システムによりパイロットと同調し操作するという従来にない操作法を利用
# しているが、その反面パイロットにはある種のＤＮＡ適合が求められ、その適合で基準
# に満たないものは動かすことさえ出来ない。
# 　これらの極端な制約や高価な維持費などにより軍内部ではソニックダイバーに対し早
# くも有用性を疑問視するものもいる。しかし、現状では過去にワーム駆逐に使われた大
# 量破壊兵器の代替となる唯一の兵器であることは間違いなく、ソニックダイバーの導入
# に積極的なものも少なからずいる。
# 　また、兵器として以外で見た場合、その小型さやパワーから災害救助用のレスキュー
# マシンとしても威力を発揮できる。

# 　Ｇシリーズの型番(例：零神のA2K0／G2K0)は、／の前に書かれたものがＯＶＡのもの
# だが、ＴＶシリーズからは最初の英字と数字がG2で統一された。また、後のKやNはエン
# ジンを開発した社名を表し、その次の数字はエンジンを換装した回数を示している。


## A2K0／G2K0 飛行外骨格 零神
#
# 　Ｇシリーズの三機のうちの一つ。パイロットは桜野音羽。
# 　近距離戦闘力を重視した機体で、機動力は三機の中で最も高い。その速度は、試作段
# 階でさえ有人飛行で上昇力の世界レコードを持っていたF-15ストリークイーグルの四倍
# 強の早さで遷音速、超音速に達するという極めて高いものだった。また、反応速度もそ
# れに準じＧシリーズ中最速だが、後に反応が過敏すぎるという音羽の意見から下方修正
# されることになる。
# 　こうした高い機動力を持つ反面、余剰なパーツを省いているため防御力は低く、内蔵
# 火器は肩部の２連ビーム砲のみと貧弱なものになっている。
# 　Ｇシリーズの中で最も初期に建造されているため、『濃く』なっている。

## ワーム ####################################################################
#
# 　侵略機械兵器群（Weapons Of Raid Machines＝W.O.R.M）。別称、機械細胞群。
# 　人類に対して敵性のある自己増殖を繰り返すナノマシンの集合体。ワーム単体では
# 大きさ３０センチ（ＴＶ版。ＯＶＡでは１インチ）ほどで特に害はないものの、それ
# が集合することで巨大化する。また、その数が多くなればなるほど能力が向上し、そ
# の大きさはＳ～Ｅの基本６段階に、＋－を加えた合計１８段階のレベルで段階分けさ
# されている。
# 　過去の大戦ではその自己増殖による再生能力や物理的攻撃力の他に、電子誘導を無
# 効化する電子妨害によって、当時はその多くを無人兵器に頼っていた人類を苦しめ、
# 最終的に大量破壊兵器（明言されていないが、製作者サイドの発言では核との事。Ｏ
# ＶＡ、小説では大質量破壊兵器）によって駆逐したと考えられている。
# 　巨大化した際の形状は初めて認識した生命体に似るという特性があり、海洋で発現
# することが多いため海洋生物によく似た姿のワームが多く見受けられる。
# 　普段は自然発生エネルギーを吸収して増殖するが、巨大な姿となったワームは人工
# のエネルギー施設、たとえばコンビナートや発電所などを襲撃し、エネルギーを得よ
# うとする。
# 　また、高度な知識もあるようで、状況に応じて退避を行い、自らを強化するといっ
# た手段も用いる。


## 桜野優希(ワーム)
#

## 【ＤＮＡ適正】
# 　ソニックダイバーに乗るために必要なＤＮＡ適正。
# 　その特性とは、アイーシャ＝クリシュナムのＤＮＡに近似である事。
# 　全世界の小学校で血液検査を行って調べている。

## 【Ｇシリーズ】
# 　日本で開発されている「God=神」の名を冠したソニックダイバーのこと。
# 　現在、零神、風神、雷神の三機が存在する。
# 　また、パラレルではあるが龍神、烈神、王神という機体も存在する。

## 【ＭＯＬＰ】
# 　Maneuver OverLap Percentage（機動一致率）の略称。
# 　ソニックダイバーとパイロットの同調の度合いを示す数値として用いられる。しか
# し、急激な上昇や下降が起こることもあり、謎の多い値でもある。

## 【ＭＶソード／ＭＶランス】
# 　ソニックダイバー用の武装。接触した対象物の分子を振動させて破砕することで切
# り裂く近接武器。

## 【五年大戦】
#
# 　ワームが人類を殲滅するために起こし、大質量破壊兵器の使用により収束するまで
# の大戦の通称。なお、これは小説版での名称であり、ＴＶでは2071～2074の四年間十
# 年前、ＯＶＡ及び小説では2079～2083の五年間三年前と、若干の違いがある。
#
# ・ＴＶ
# 　ワームの地球規模での人類掃討作戦により人類は窮地に立たされた。それを打破す
# べく人類は大量破壊兵器を用い、陸地の50％と人口の三分の一を失いつつ辛くも勝利
# を収めた。その際、人類は主たる戦闘員であった20～30代の若い男性の実に九割を失
# ってしまった。
# 　なお、勘違いされやすいが、ＴＶ版ではあくまで10年前に20～30代だった男性を失
# っているので、物語開始現在ほとんどいないのは30～40代の男性である。
# 　現在では復興もだいぶ進んでいるが、ところどころに傷跡が見られる。
#
# ・ＯＶＡ、小説
# 　基本的にはＴＶと同じだが、ワーム殲滅に使用したのは大質量破壊兵器という、大
# 陸の地形、海溝の形さえ変えてしまうという超破壊兵器。また、終戦はつい三年前な
# ので、こちらでは20～30代の男性がほとんどいない。
# 　終戦間もないため現在は復興を進めている最中である。

## 【スプレッドブース】
# 　ナノスキンジェルを塗布する専用の装置。
# 　基本的には備え付けの大型の装置なのだが、持ち運びが容易な小型のものも存在す
# る。また、適正な電力がないとエレベータが起動できなかったり、ナノスキンジェル
# の持続時間が短くなったりする。

## 【デルタロック／クアドラロック】
# 　三機（クアドラロックの場合四機）のソニックダイバーの連携により対象を重力場
# の内部に捕らえ、集中砲火を浴びせる必殺技。ワームに対して通常攻撃は効果が薄い
# が、ＭＶソードを介しパイロット情報を送り込むことでホメロス効果を引き起こし、
# 通常兵器で殲滅可能な状態にしている。
# 　また、デルタロックの三機というのは、重力場を形成するために必要な二次元平面
# を作るうえでの最小の数値である。
# 　大量破壊兵器の代替となる唯一のワーム殲滅兵器。

## 【ナノスキンジェル】
# 　スプレッドブースにて塗布されるナノマシンを含むピンク色のジェル。
# 　塗布された後、一定の電圧をかけることで活性化し、パイロットへ浸透。酸素の供
# 給、耐Ｇ、耐圧、耐熱、耐冷、耐風とあらゆる生命維持に効果を発揮する。このため、
# パイロットはほぼむき出しのまま超音速機動にも耐えることができ、高Ｇで旋回して
# もまったく影響を受けない。
# 　しかし、電圧をかけてからの活動限界は21分32秒であり、それを超えるとホメロス
# 効果による自壊作用によって効果がなくなってしまう。そのため、活動限界を過ぎれ
# ばパイロットは普通の人間と変わりなく、超音速機動などには耐えられなくなる。
# 　新緑の微香があり、アレルギー反応を示す人もいる。

## 【ネスト】
# 　太平洋上に浮かぶワームの本拠地であり、生産工場。それ自体が巨大なワームであ
# り、防衛のために大量のワームを生み出して戦闘を行った。

## 【ホメロス効果による自壊作用】
# 　ナノマシンと人との融合はナノマシンに悪影響を及ぼし、組成の維持を困難にし、
# やがて自壊を始めてしまう。このナノマシンと人の融合が及ぼす悪影響を「ホメロス
# 効果」、その後に始まる自壊のことを「ホメロス効果による自壊作用」という。
# 　この効果により、ナノスキンジェルは21分32秒しか効果を維持できない。また、こ
# の効果を利用してワームの組成を崩壊させるのがデルタロック前に行うＭＶソードに
# よる攻撃である。

## 【ナノシステム治療プログラム】
#
# 　クリシュナム博士が開発した難病治療プログラム。
# 　ナノマシンを体内に浸透させて細胞自体を強化し、さらにナノスキンを形成して生
# 体維持を行い、被験者に患部があれば自動的にそこを修復するプログラム。
# 　生体ナノシステムを利用した技術で癌をはじめ難病治療にも有効であり、ナノマシ
# ン自体はホメロス効果により一定時間で自動的に自壊し、体内に吸収、排出されるた
# め無害である。

## 【ワームが人類に敵対する理由】
#
# 　ワームの正体は地球を一個の生命体とみなして巨大な治療システムを構築したナノ
# システム治療プログラムであり、人類を地球に有害なものと判断し、プログラムに従
# って患部の治療、つまり人類の根絶を目指している。
# 　そうなってしまった理由は、ナノシステム治療プログラムは規模が大きくなると自
# 己判断能力が強化されるという特性があり、ネットワーク上に流出したプログラムが
# 肥大化し、勝手に治療を始めたためである。
# 　なお、大量破壊兵器（大質量破壊兵器）の使用により一時は退けられたと思われて
# いたが、それは兵器の使用が人類の自滅行為だとワームが判断して活動を停止したた
# めであり、人類が復興し始めると同時にワームも再び活動を開始することになった。

## 【アイーシャ】
#
# 　ワームはナノシステム治療プログラムであり、その開発に深く関わっていたアイー
# シャはいわばワームにとって母と呼べる存在といえる。
# 　同時に、アイーシャにはナノマシンと人体の完全なる融合「人機一体」が可能であ
# り、そのシステムを元にソニックダイバーは開発されているため、ソニックダイバー
# にとってもアイーシャは母と呼べる存在である。
# 　つまり、ソニックダイバーとワームは兄弟と呼べる存在であり、同時にアイーシャ
# と遺伝子的に近似であるソニックダイバーに適性がある人類は、ワームとの適性も高
# いということになる。
# 　また、アイーシャは上記の特異性からかソニックダイバーやワームを支配下に置く
# という特殊な力があるが、その発動後は極端に体力を消耗してしまう。
