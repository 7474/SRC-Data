# ゴルディーマーグおよびマーグハンド開発前の、19～20話再現用。ゴルディ加入後は必ず削除してください。
#
# 衝撃緩衝ブロックたるマーグハンドがないため、使用にはＨＰの30%を失うというペナルティがあります。
# また、対象の核を保護するハンマーヘル・アンド・ハンマーヘブンの過程が行えないことから、
# 単独のハンマーには浄属性がつきません。
# ＨＰを消費せず、確実に浄解したい場合には、合体技のＤフォーメーションを使用する必要があります。
#
# ゴルディーマーグ開発前という、きわめて限定的な状況の再現であるため、
# Ｄフォーメーションの戦闘アニメやダイアログは大河長官仕様のみ。
# エヴォリュダー版のガイには、そもそも対応ダイアログが存在しません。
#
# なお、マーグハンド未装着だったＴＶ最終話の再現にも一応使えますが、
# 合体と違い当然にはゴルディが乗らないため、できればシナリオ側でサポートとして乗せてやってください。
# とはいえ、ハンマーが破られた後は当アイテムも削除することになるでしょうから、
# 本当にこだわる方以外は、普通に合体→分離させる方が楽だろうと思います。

# 外伝小説『獅子の女王』にて使用。
# 中間子を対消滅させる反物質フィールドにより、いかなる物体をも易々と切り裂く巨大カンナ。
# ゾンダリアンが本格的な活動を開始する前、2005年3月には開発が完了していたが、
# その破壊力でゾンダーロボの核＝素体となった地球人まで粉砕してしまうため、実戦投入を断念。
# 以後、Ｇストーンを取り外して封印されていた。
# 対Ｇ－ギガテスク戦において使用されたのは、相手が無人で人命を考慮する必要がなかったことと、
# ゴルディオンハンマーが対ＺＸ－０６戦のオーバーホールを終えていなかったという、
# 複数の要因が重なったためである。
#
# 設定画に「１回分のエネルギーが入ったカートリッジ」が８つ描かれていることから、
# 弾数８と破格の連続性能を誇ります。
# ですが、小説では一度の戦闘で３回使用しており、一撃で敵を削りきれるものでもないようです。
# おそらく、ある程度の連続使用が前提となったツールなのだと思います。
#
# そこで、50％の確率で再攻撃を行うようにしてみました。
# 他の必殺技に比べ攻撃力を抑えてありますが、場合によっては数値以上のダメージを叩き出す危険なツールです。
# また、弾の減りが意外と早い、という効果もあります。
