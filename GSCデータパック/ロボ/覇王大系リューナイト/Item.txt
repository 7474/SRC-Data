# 『魔法石』
# いくつかのリューの精霊石を一つにまとめたもの。それほど大きくはないが
# 凄まじい力を持つ。マジルカ一族の秘宝。グレーガスとの最終決戦において
# アデュー達に貸し出された。その際、セイルも高速飛行艇で付いてきた。
# なお、貸し出しの条件としてパッフィーとセイルの結婚があり、非常事態
# なのでパッフィーはこれを了承した。
#
# データ的には、リュー達をパワーアップさせるアイテムとしています。
# 通常形態と精霊石形態の中間くらいになるように設計しました。
# これの加護があれば、強大な魔族と戦っても生き残ることが可能です。
# また、リューの精霊石から作りだした物のためドゥームには効果が薄い、
# というかなり無理のある設定を採用しています。味方ドゥームが
# 強くなりすぎるのを防ぐためです。(キルガイン、強すぎ……)
# リュー用とドゥーム用、２種類を用意することにしました。
#
# この魔法石を使う、使わないはシナリオ制作者の自由です。
# バランスを取るための手段の一つとして用いて下さい。
