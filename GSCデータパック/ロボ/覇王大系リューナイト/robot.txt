# 【参考用データ】
# 公式データ化するには問題のある機械都市エルドギアを、
# 参考用としてここにおきました。

# 機械都市エルドギア
# 機械都市エルドギア, 要塞(ホワイトドラゴン専用), 1, 2
# 空陸, 0, XL, 65000, 250
# 特殊能力
# バリアLv4=エルドギア・バリア装置
# グレーガス並み, 凄く多い, そこそこ硬い, ６０程度？
# AAAA, RK_Eldogear.bmp
# 自動機銃,       結構痛い,               1, 2,+25, 99, -, -, AAAA, +10, 銃
# ミサイル群,     相当痛い,               2, 5, +0, 99, -, -, AAAA, +5, 実
# 中性子ビーム砲, ドゥームも一撃,         1, 4,+15, -, 連射可能, -, AACA, +10, Ｂ
# 攻撃衛星砲,     ＋魂でグレーガスも一撃, 6, 20,+0, -, 数発, -, AAAA, +0, Ｃ消ＢＭ投L3
# 自爆型攻撃衛星砲, 上に同じ,             1, 3, +0, -, 数発, -, AAAA, +0, ＣＢ自Ｍ全
#
# 機械都市エルドギア(動力不足)
# 機械都市エルドギア, 要塞(ホワイトドラゴン専用), 1, 2
# 空陸, 0, XL, 65000, 250
# 特殊能力
# バリアLv2=エルドギア・バリア装置
# 上の７割程度？, 上の半分程度？, 上に同じ, ６０程度？
# AAAA, MachineCity-Eldogear.bmp
# 武装は上に同じ
#
# イベント限定のユニットです。動力不足の状態でグレーガスと戦い、
# 最終型攻撃衛星砲で自爆してグレーガスを仕留めます。が、復活の呪文で
# グレーガスは甦ってしまう(ただし復活直後で弱ってる)のです。
# 実はこの時アデュー達も爆発の効果範囲内にいましたが、ビームが直接当たった
# 訳ではないので、イズミとデスローザのバリアで防ぎきることができました。
# 都市だけあって動けないわけですから、完全イベント専用でしょう。
# ちなみに動力が不足しているのは、異世界カイオリスへ行った
# アデュー＆パッフィーとの通信にエネルギーを使いすぎたせいです。
#
# ・攻撃衛星砲について
# エルドギアに太陽エネルギーか何かを供給している装置です。
# このエネルギーの供給をストップし、攻撃エネルギーに変えて
# 衛星軌道上から敵を狙い撃ちにする、おそらくリューナイトの世界で
# 最も破壊力の高い(魔王以上！)兵器です。ただし、敵がエルドギアに
# 近すぎると………エルドギアを巻き込むことになります！
