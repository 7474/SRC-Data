『黒き月』
# 正体は自意識を持った自動戦闘衛星。
# 宇宙の先史文明『ＥＤＥＮ』が、対立する種族『ヴァル＝ファスク』との戦いのさなか、
# 彼らを打ち破るための決戦兵器として『白き月』と共に開発された。
# 『不確定要素を徹底的に排除し、常に安定した戦力を生み出す』という設計思想の基、
# その戦力のほぼすべてを自らが作り出す無人艦によってまかなっている。
# エオニアと接触し、『白き月』の居場所を知ったことで、
# 更なる力を得るためにエオニアを利用して『白き月』と融合しようとするが、
# 最終的にはエルシオールのクロノブレイクキャノンによって破壊される。
# 
# なお、設計思想の違う二つの月が作られたのは、
# 両者を戦わせることによって戦闘データを蓄積させ、同時にその優劣を競わせるため。
# (言うならば試作機のトライアルのようなものである)
# 最終的な勝者が敗者を吸収融合することにより、
# 勝者の設計思想のもと、両者の特長を最大限に生かした決戦兵器を作り上げ、
# それをもって『ヴァル＝ファスク』との決戦に挑む……というのが当初の計画だったが、
# 時空震の発生により両方の月を管理運営していた文明『ＥＤＥＮ』は壊滅、
# 物理的な距離が離れたこともあって
# 二つの月はそれぞれの思考ルーチンによる独自行動を開始する。
# その結果、『ヴァル＝ファスクを倒すこと』を最優先させた『黒き月』は
# 『ＥＤＥＮ』の生き残りであるトランスバールの住人さえも戦闘データ蓄積の対象と認定、
# 結果トランスバールに未曾有の危機をもたらしてしまう。
# 一方、『人類を救うこと』を最優先させた『白き月』は、積極的に人類にかかわり
# 蓄積されたデータを惜しみなく提供するが、
# その過程で『ヴァル＝ファスク』という存在自体を忘れてしまうことになる。
# このような状況で、両者の決別は半ば必然だったと言えよう。
