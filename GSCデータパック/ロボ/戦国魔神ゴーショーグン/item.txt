#######強化パーツ########

ケン太のポケコン
ケン太のポケコン, けんたのぽけこん, 汎用, 強化パーツ
特殊能力
ＣＴ率強化Lv2
パイロット能力付加="修理Lv2=(ポケコン分析), 1" (真田ケン太)
0, 0, 0, 0, 0
*全ての武器のＣＴ率が１０％上昇する。また、真田ケン太がメインパイロットの
*ユニットがこれを装備すると、修理装置での回復量が２０％上昇する

#ケン太愛用のポケコンで、ジェッターエースと連動して起動させたり、ファザーからデータを受け取って、
#損傷したメカの修理に使用されていました。
#「第二次スーパーロボット大戦α」での効果に加え、ケン太本人が装備すると、修理装置の効果が上昇します。

ＯＶＡのコロッケ
ＯＶＡのコロッケ, おばのころっけ, 汎用, 強化パーツ
特殊能力なし
0, 0, 0, 0, 0
===
ＯＶＡのコロッケ, 解説=ＳＰを20回復 ＳＰ回復Lv2 再行動, 0, 1,   -, -, (アイテム)
*パイロットのＳＰを20回復
*使い捨て

#ＯＶＡの作ったコロッケを褒めたら、ＯＶＡが調子に乗って、グッドサンダーの面々は、
#二週間も毎食コロッケ漬けの憂き目にあったという、いわく付きのシロモノです。（笑）
#「第二次スーパーロボット大戦α」と同一の効果です。

ケルナグールフライドチキン
ケルナグールフライドチキン, けるなぐうるふらいどちきん, 汎用, 強化パーツ
特殊能力なし
0, 0, 0, 0, 0
===
ケルナグールフライドチキン, 解説=ＳＰを10回復 ＳＰ回復Lv1 再行動, 0, 1,   -, -, (アイテム)
*パイロットのＳＰを10回復
*使い捨て

#ＯＶＡの作ったコロッケを作成したので、ドクーガ製のアイテムも作成してみました。（笑）
#ケルナグール・フライドチキンは、ケルナグール食品の社長でもある、ドクーガの
#ケルナグール司令官が、特に力を入れているファーストフードチェーンで、ゴーショーグン世界
#では、かのケンタッキー・フライドチキンと、世界のフライドチキン業界を二分する存在です。
#「宇宙中継これがドクーガだ」において、ドクーガの存在を世界に暴露されたドクーガが、一転
#して世界にドクーガ関連のＣＭを流し始めた時期には、ケルナグールはヨーコ夫人と共に、
#自らケルナグール・フライドチキンのＣＭに出現していました。
#量販品なので、ＯＶＡのコロッケには効果が劣ります。（笑）

########グッドサンダー第三段階以降表現アイテム########

ビムラー瞬間移動装置
瞬間移動装置, しゅんかんいどうそうち, 汎用(グッドサンダー基地専用), 非表示
特殊能力
テレポートLv5=ビムラー瞬間移動 200
緊急テレポートLv10=ビムラー瞬間移動 4 200 100 手動
ビムラー瞬間移動=解説 "攻撃を受けた際に回避を選ぶとテレポートし、攻撃を回避。また、テレポートを行い、移動力+5で地形を無視して移動。200ＥＮ消費。"
0, 200, 0, 0, 0
*ビムラーが第三段階に進化し、グッドサンダーは地球上のいかなる場所にも
*時間制限無しに瞬間移動可能になった。

#ビムラーが第三段階以降に進化した時のグッドサンダーを表現するアイテムです。
#機体のＥＮが更に上昇し、瞬間移動が可能になります。
#本来は瞬間移動能力は戦闘中に使用する事はほとんどありませんが、瞬間移動能力を組み込みたい場合に。
#尚、瞬間移動は第三段階で地球上のいかなる場所にも時間制限無しに瞬間移動可能。破壊エネルギーは
#それまでの太陽系全域破壊規模から、除々に減少。第四段階以降だと、瞬間移動を繰り返す事で、外宇宙
#への移動も可能なレベルになり、破壊エネルギーは消失し、ビムラーは無害なエネルギーとなります。
