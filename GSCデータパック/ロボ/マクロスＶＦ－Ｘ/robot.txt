# 　ゲーム中、ヴァルキリー隊が最初に使用しているのは、通常のＶＦ－１ではなく、これらの
# 機体です。通常のＶＦ－１シリーズは、登場しません。
# 　ゲームでの扱いからは、性能的にはＶＦ－１シリーズとさほど変わりない物と思われますが
# （違いと言えば、スーパーパックに２連装ビームキャノンが付いている程度です）、それでは
# データ化してもあまり意味がない、使われないデータとなって終わるのではないか、と言う
# 指摘を受けた為、路線変更しＶＦ－１Ｅｖｏシリーズを、「シナリオの中盤まで使えるＶＦ－１」と
# 言う位置付けにし、名前通りＶＦ－１の発展型として、基本性能を向上させたデータにしてあります。
# 　ある意味、ファーストマクロスの輝やフォッカーを、ＶＦ－１以外に乗せたくない、と仰る方
# 向けのユニットです。
# 　ランク的には、素の状態ではＶＦ－４に多少劣る程度、スーパーパック装備時にはＶＦ－１１の
# 素の状態に迫る程度に調整されています。
# 使用に関して注意すべき点等はありません。うまくシナリオに組み込んで使ってあげて下さい。
# 　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　文責：川崎貴之
