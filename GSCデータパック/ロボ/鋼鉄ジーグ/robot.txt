#─────────────────────────────────────────────
# ビルドベース
#─────────────────────────────────────────────
# 鋼鉄ジーグ
#
# ・Ｓサイズスーパー系。避けて当てて少しだけ耐える。
# ・新サイボーグへの強化後を鋼鉄ジーグ、それ以前を前期型として設計。
# ・ビッグシューターから半径3マス以内にいると、武器が増えたり換装できたり。
# 　(美和をサポートとして同乗、またはアイテム装備でも可)
# ・撃墜された際ジーグヘッドにパーツ分離→再ビルドアップ可能。
#   ただし、事実上、無限にＨＰを回復できてしまうため手動分離は不可とする。
# ・パーンサロイド以外の換装は運動性減少。追加パーツの分重いものとみなす。
# ・ＳＲＷと違い、マッハドリルは別形態。移動可能な地形が明らかに異なることと、
#   パーンサロイドやスカイパーツ登場まで、まったく飛行できないのも不便であるため。
#   主に前期型の救済案。専用パーツほどの適応がないと判断し、運動性減少幅は少し多め。
# ・マッハドリル射出後は腕なし形態に変形。
#   ビッグシューターから半径3マス以内に入ることで、自動ハイパーモードで通常形態に復帰可能。
# ・スカイパーツ、マリンパーツの変形はオミット。画像変更で対応。
# ・パーンサロイドはハイパーモード扱い。ビッグシューターには積載できない模様。
#   支援メカがどこからともなくやってくるのは『よくあること』として割り切る。
# ・将来的には鋼鉄神との共闘もありうるため、ダブルスピンストームに対応しておく。
#   これはパイロット側で指定。
#
# ナックルボンバー三連射は、原作初期の再現シナリオではDisableを推奨します。
# (それ以前にも手が生える描写はありますが、はっきり連射するようになるのは第12話以降)
#
# 攻撃力比較
# マッハドリル＞アースドリルシュート＞ジーグブリーカー＝スカイジーグドリル＞ジーグバックラー＞スピンストーム
#
