# ACE COMBAT 3 electrosphere

が台頭し、国家という枠組みが意味を成さなくなった2040年。両者の緊張が
高まる中、特殊部隊ＳＡＲＦに無期限待機命令が下された。

【ＵＰＥＯ】：Universal Peace Enforcement Organization　の略称
　ＮＵＮに所属する治安維持対策機構。「ユーピオ」と読む。
　民族や国家の枠組みを超えた地球規模での平和維持、軍事的、経済的な制
裁活動が本来の役割だが、近年まで平和維持委員会はゼネラルリソース
（以後ＧＲ）の傀儡議員で構成され、ＧＲが表立って制裁を加えられない企
業に対して合法的にコントロールするためなど、様々なことに利用されてい
た。そのため、企業の利益優先の政策や、軍事力の無さから『張子の虎』と
揶揄されてきた。
　しかし、2037年にガブリエル＝Ｗ＝クラークソンがＵＰＥＯ治安維持機構
代表着任後、独自の『平和主義精神』政策に則りＵＰＥＯの体制変革を推し
進め、本来の役割が明確化され現在に至る。
　また、建前上企業の利益に加担しないということで独自に兵器開発をして
きたが、最近ではＧＲ派とニューコム派の議員の派閥争いが耐えず、双方か
らの武器供給が開始されている。

・ＳＡＲＦ　：Special Armed Response Force　の略称
　ＵＰＥＯ所属の特殊部隊。「サーフ」と読む。
　紛争を鎮圧する軍事的治安維持を目的とする部隊だが、実際は戦闘経験の
ない若手パイロットで構成された、お飾り的な部隊となっている。実際、Ｕ
ＰＥＯ内部でも末端に位置する組織であり、軍事介入などの決定権はほとん
ど無いといっていい。さらに、『ＮＵＮによる侵略』というイメージを与え
ないよう、高価、高性能な武装は大量に使用することができないというジレ
ンマを持つ。『張子の虎』という揶揄を体現したような部隊といえる。
　ただ、最近はＧＲとニューコムの対立の激化により、各社からの武器供給
によって戦力の増強が行われている。

・ＮＵＮ　：Neo United Nations　の略称
　国際平和の維持などを目的とするＵＰＥＯの母体組織。国連のこと。
　フェイス・パーク紛争以降緊張が高まるＧＲとニューコムの仲介役が現在
の主な仕事である。

・ＥＡＰＮ　：Euro-Asia Public Network　の略称
　ＵＳＥＡ国営放送局を指す。2003年設立。「エーペン」と読む。


【General Resource LTD.】
　あらゆる産業の中核を握る国家の枠組みを超えた巨大資本の統合体。
　資本主義社会が発展するにつれて経済市場のグローバル化が進み、吸収合
併を繰り返した果てに生まれた世界で一番巨大な多国籍企業。その活動はあ
りとあらゆるジャンルにおよび、『ＧＲ社のロゴが入っていないモノは無い』
とまで言われる。つまり、一つの企業に生活のすべてを握られている状態に
なってしまっている。その力は法律を凌駕し、国家を単なる国境を区分する
だけのものにしてしまった。
　最近では大陸間道路やジオフロント計画に着手しその健在振りを示してい
るが、そういった開発による環境破壊は地球環境保護団体グローバルウェイ
の批判の対象となっている。
　しかし、近年のニューコムの急成長には危機感を抱いており、経済制裁が
軍事制裁にまで発展してしまっている。

・ＧＲＤＦ　：General Resource Defense Force　の略称
　ＧＲ社の防衛部隊。
　実戦能力の高いパイロットで構成された部隊。その多くは大学卒のエリー
トだが、キース＝ブライアンのように高卒でも実力のみでのし上がることが
可能。他の組織と違いもっとも軍隊らしい軍隊といえる。
　最近まで対抗できる組織が無かったために兵器の新規開発などは行われて
おらず、保守的になったＧＲ上層部の考えから新型兵器の開発も足踏みして
いる状態である。

・ＧＢＳ　：General Broadcasting System　の略称
　ゼネラルリソース系列スフィア民営放送局を指す。


【Neucom Inc.】
　元ＧＲの科学者によって創設され、宇宙産業、ナノテクに長ける、ＧＲに
対抗しうる現在唯一の企業。「ニューコム」と読む。
　2030年のＧＲ社からの開発スタッフ大量移籍を皮切りに、優秀な人材を積
極的に集め、事業内容を拡大させた。それにより、当初の宇宙開発における
先駆的存在から、ＧＲが苦手とする柔軟性が求められるテクノロジー分野に
おいてＧＲを脅かす、もしくは抜き去るほどの成長を遂げた。
　しかし、その裏で非人道的ともとれる実験、開発を繰り返していることが
非難の対象となっており、それはニューコムが独自の軍部を所有してからは
一層激しいものとなった。ただ、ニューコムがここまで非難を浴びるように
なったのは、2036年から本格化したＧＲによる制裁でＧＲ系列やＵＰＥＯ系
列のメディアを操作した『ニューコム＝悪』という印象付けが影響している。
　もっとも、当のニューコムは制裁に一切応じず、二社間の企業抗争は本格
化している。
　また、2039年にはＧＲによる制裁の中で海上浮遊都市メガフロートを完成
させ、非難されながらも開発した製品が使用されていることから一部では
「不可能を可能にする」とまで囁かれるほどになっている。
　新世代AIや連想記憶メモリの開発等、コンピュータ技術の研究開発を担当
する情報環境開発事業部「ニューコム・インフォ」、航空宇宙産業を担当す
る宇宙環境開発事業部「ニューコム・スペース」、バイオ・テクノロジーや
ナノテクノロジーを担当する生命環境開発事業部「ニューコム・バイオ」、
ＮＯＣ事業やエレクトロスフィアを世界規模で構築するなど通信事業を担当
する通信環境開発事業部「ニューコム・スフィア」、ニューコムの対外的広
報・外務交渉・報道や出版による大衆操作を行う広報外務部「ニューコム・
パブリッシング」などの部署に分けられている。
　
・ＮＥＵ　：Neucom Emergency Unit　の略称
　ニューコム所属の非常事態配備機構。
　エリートの中のエリートで構成されており、兵器の開発、研究が盛んで機
体の性能が最も高い組織。他の組織と違い研究者の発言権が大きく、人体実
験まがいの研究さえ許容されるのが大きな特徴となっている。そのタブーの
少なさから、ＧＲも存在を大きなものとして捉えている。
　また、兵器は性能だけではなく機能美にこだわったデザインで、質実剛健
なＧＲＤＦのものとは大きく違っている。

・ＥＡＳＡ　：Euro-Asia Speace Administration　の略称
　ニューコムの前身で、元政府宇宙管理機構。
　慢性的な資金難により民営化。その後大手航空会社吸収によって宇宙開発
の先駆的存在となった。

・ＮＶＳ　：Neucom Visual Server　の略称
　ニューコム系列スフィア民営放送局を指す。
　電脳空間を利用したテレビ放送を、他の放送局に先駆け開始した。


【ＯＵＲＯＢＯＲＯＳ】
　総人類電脳化を目指すクーデター組織。リーダーはアビサル＝ディジョン。
　ＥＣＭ空中空母スフィルナを母艦とし、レナのナイトレーベンやディジョ
ンのオーロラなど超高性能機を筆頭に、各企業、組織から離反した者が持ち
出した機体が主戦力となっている。また、持ち出した機体のほとんどが最新
鋭の高性能機であり、少数ながらも精鋭揃いとなっている。


【グローバルウェイ】
　地球環境保護団体。「地球の道が、我々の進むべき道である」をスローガ
ンに、環境破壊のみならず化学実験、人体実験などにも反対している。

【エレクトロスフィア】：Electrosphere
　電脳空間の意。『electro：電気的な』と『sphere：粒子の集まる球体』を
あわせた造語。

【ＥＮＳＩ（エンジー）規格】：Electro-Neuron-Synapse-Interface
　パイロットの神経細胞とコンピュータを接続する回路のバス規格。
　これにより、兵器を精神操作することができる。

【オプト・ニューロン】：Opto-neuron
　最新の人工神経網であり、光速神経網とも言う。
　従来の規格品より高度な操縦が可能となったが、直接神経に植えつけるた
め衝撃により脳死や神経障害などを引き起こす恐れがある。
　また、上記の理由により取り外しできない（＊１）。

【サブリメーション】：Sublimation
　電脳化の意。本来は固体が気体に昇華するという意味だが、そこから人が
肉体を捨て、データのみの存在になることという意味に派生した。
　ＯＵＲＯＢＯＲＯＳが目指す人類の進化の形態。

【シルバーストーン病】：Congenital E-lrradiation Syndrome
　レナの罹っている先天性の病気。正式な病名は『先天性Ｅ照射症候群』。
太陽光に含まれる『E-ray』を浴びることができなくなる。
　通称は発見者のアントニオ＝シルバーストーン博士から。

【ターミナルフェイズ】：Terminal Phase
　ウロボロスの目指す人類総電脳化計画。

【デッドコピー】：Dead-Copy
　複製人格の意。人の脳をコピーして作った人格の俗称。

【フェイス・パーク紛争】
　ＧＲがニューコムに制裁を加えたことに端を発し、2040年に勃発した紛争。
これにより、ＧＲとニューコムの対立は決定的なものとなった。
　その際、両者の間を取り持ち停戦協定までこぎつけたのがクラークソン代
表である。
　名称は紛争が勃発した地点から。

【メガフロート】
　ニューコムの海上移動都市。軍事施設を中心に多くの設備がある。

【ＤＯＥ軍事研究所】
　ゼネラルリソース空軍の極秘研究施設。
　2030年に事故に見せかけて爆破される。

【Ｏ．Ｓ．Ｌ】：Orbital Satellite Laser　の略称
　衛星軌道上レーザーの意。衛星軌道の軍事衛星からのレーザー攻撃。

【Ｒナンバー】
　ニューコムが開発した戦闘機の総称。Ｒは「革新的」を意味する
「Radical」から。

【Ｎｅｍｏ】

　名前の意味はラテン語で「誰でもない」。


【Ｎｅｍｏに関する謎】
・ゲームのディスクラベルに書かれた『2030 S.O.C Project “NEMO”』とい
　う一文。
・Ｎｅｍｏに関連するかどうかは今のところ不明だが、グローバルウェイが
　糾弾していた元・ゼネラルの科学者達（現在のニューコム社員）による非
　人道実験と『Ｓ.Ｏ.Ｃ文書』の内容。
・サイモン博士が「機械に人格を持たせるのは、神でもなければ“無理”。
　但し、実在人物の人格をＡＩ等にコピーする事は可能」と断言していた。


【アビサル＝ディジョン】
　ディジョンはＤＯＥ軍事研究所爆破事件の際に、死亡している。そのとき
にデッドコピーとなったと思われる。そして、恋仲であったヨーコ＝イノウ
エの死亡を直視できない彼は、革命の決意をすることになる。
　彼が４０を過ぎてもエースでいられたのは、肉体を捨てた存在（デッドコ
ピー）となっていたため。


【ＤＯＥ】：Darkness of Enigma　の略称
　ゼネラルリソース社において極秘に行われていた軍事的開発計画。
　判明している主な研究内容は以下の通り。
・ナイトレーベン　：新型戦闘機の開発
・オプトニューロン：従来型のＥＮＳＩシステムに代わる情報伝達媒体の研究
・サブリメーション：人格の電脳化、イノウエ博士が基礎理論を構築
　また、最終的には人類滅亡後の研究を目指すという説もある。


【ＤＯＥ施設爆破　その後】
