
〇ダイターン３武装解説

#　ダイターンクロスダート：         　両手の平から発射するレーザー網。
#                                   手首から先を飛ばしてから発射も可能。
#　ダイターンスナッパー：           　両腰のホルスターに収められている鞭。
#                                   二段階に延び、手錠や投げ縄にも変型する。
#　ダイターンファン：               　両足に収納されている扇型の武器。自由度
#                                   が非常に高く以下のような使用例がある。
#
#                                   　・畳んだ状態で叩いたり、投擲する。二刀流も有り。
#                                   　・開いて扇ぐ。二枚で強風を起こすと
#                                  『必殺ファン返し』となる。
#                                   　・開いて投擲する。連結して円盤状にして
#                                   投擲する事もある。
#                                   　・畳んだ状態、もしくは開いた状態で二本を
#                                   柄で連結して、ヌンチャクにする。
#                                   　・展開した一枚、もしくは二枚をシールド
#                                   代わりに使用する。
#                                   　・展開した二枚を連結し、手持ちのシールドにする。
#
#                                   　どの武装も他の武装の劣化版にしかならないので、
#                                   ここでは畳んだ状態で打撃武器にする場合のデータだけ実装。
#　ダイターンミサイル：              　腰に一門装備されているミサイル。
#                                    万丈にまで威力が無いと言われている。
#                                    作中でホーミング兵器としての特徴を
#                                    強調する描写有り。 
#　サンレーザー：                    　作中や設定にもこの名称は見られない。
#                                    それらしい武装の使用例は、第十話で頭に
#                                    振り下ろされた剣をカウンターで破壊した時のみ。
#                                    その時は「サンアタックのレーザーの輝きは、一瞬でも
#                                    これだけの威力がある」と呼ばれている。
#　ダイターンジャベリン：            　両足に収納されている槍。２００mまで伸びる。
#　ダイターンキャノン：              　両足の底にある大砲。砲塔を向ける方向の
#                                    自由度は高いが、体勢が不自然だったり、
#                                    両門撃ちを行うためにはジャンプする必要が
#                                    あるせいか、主に牽制に使われた。
#　ダイターンウェッブ：              　両腕の十字を取り外して投擲する手裏剣。
#                                    また、腕に付いたままの状態で円形に展開し、
#                                    「ガード・ウェッブ」という盾にもなる。
#　ダイターンザンバー：              　両足に柄が収納されている、斬馬刀型の剣。
#                                    柄からレーザーの刀身が出現した後、実剣となる。
#　ビッグウェッブ：                  　胸の十字を取り外して、変型させる巨大手裏剣。
#                                    ブーメランのように使用する。
#　ダイターンハンマー：              　両踝に収納されている棘付き鎖鉄球。
#                                    投擲例も多い、実は棘を射出する機能もあるが、
#                                    本編未使用。
#　サンアタック：                    　蓄積した太陽エネルギーから、パルス・イオン・
#                                    エンジンが作り出した、サンビームと呼称される
#                                    光弾を額から発射する、ダイターンの必殺技。
#                                    　サンビームにはメガノイドの使用している
#                                    装甲を劣化させる機能があり、装甲劣化後に
#                                    ダイターンクラッシュ（ドロップキック）で
#                                    ぶちぬき、とどめとする。
#                                    　ただし、これ自体が強力な破壊光線でもあり、
#                                    後半はクラッシュ抜きのサンビーム単体でとどめを
#                                    刺すようになる。
#                                    　データ上は、命中時はサンビーム単体、ダメージは
#                                    クラッシュ込みのため、Ｂ貫劣属性です。
#　サンアタック乱れ撃ち：            　第三十九話で、サンアタックを容易く回避する
#                                    相手に対し、低出力のサンアタックを広範囲に
#                                    バラ撒いて対処した技。
#                                    ２６発以上が命中し、相手は穴ボコに。
#                                    　データ上は超高命中必殺技というコンセプトで、
#                                    連属性は〇十発命中というインパクトを出すための
#                                    雰囲気付けです。
#　コンビネーションブレード：        　ＳＲＷでの、ザンボットとの合体攻撃で、
#                                    ダイターンがザンバー、ザンボットがカッター
#                                    を使用します。
#　スぺースコンビネーションアタック：　ＳＲＷでの、ザンボットとの合体攻撃で、
#                                    ダイターンがサンアタック、ザンボットが
#                                    ムーンアタックを使用します。

#　以下、直にデータに組み込んでいない能力
#
#　ダイターンスモーク：　　　　　　第二十七話で腹部のミサイル発射口から出した煙幕。
#　必殺ファン返し：　　　　　　　　第二十四話でファン二枚で扇いで強風を起こし、巨大
#　　　　　　　　　　　　　　　　キノコの胞子を扇ぎ返した技。　　　　　　　　　　　　　　　　　
#　ダイターンジャベリン先端射出：　第十五話で、ジャベリンの先端を射出して、遠隔操作
#　　　　　　　　　　　　　　　　した攻撃。
#　ダイターンザンバー二段斬り：　　第一話で、ザンバー二本を使う事で相手のロープによる
#　　　　　　　　　　　　　　　　緊縛から脱し、反撃した技。
#　ダイターンバリア：　　　　　　　第十八話で、光子に変化してダイターンを包み込んだ
#　　　　　　　　　　　　　　　　相手を弾き飛ばしたバリア。
#　ダイターン逆磁力：　　　　　　　第三話で、強力磁力板のスーパーマグネボードから
#　　　　　　　　　　　　　　　　脱出を試みるために発動させた。
#　温風：　　　　　　　　　　　　　第十四話で、凍結した街とその住人を解凍して救出
#　　　　　　　　　　　　　　　　するために、８８度に設定して口から吹き出した。
#　電子冷却：　　　　　　　　　　　第十七話で、灼熱化した相手の超高温攻撃を相殺
#　　　　　　　　　　　　　　　　しようと発動させ、自身の機体を冷却したが、効果は
#　　　　　　　　　　　　　　　　上がらなかった。　

〇ダイファイター武装解説

#　ダイターンミサイル：    　機体下部に一門装備されている。ダイターン時と同一の物。
#　爆弾：                  　作中ではミサイルを投下爆弾として使用しているため、
#                          ダイターンミサイルと弾数を共有させています。
#　ダイファイターレーザー：　機首から発射する二本のイナズマ状光線。
#　ダイファイターミサイル：　機首に二門装備されているダイターンミサイル。ノーマルの
#　　　　　　　　　　　　　ダイターンミサイルと区別するため、ダイターン３大全内で
#　　　　　　　　　　　　　使用されていた名称を使用しています。
#　サンビーム：            　機首の先端から照射される熱線。サンアタックのレーザーの
#                          バリエーションと推測されるので、この名称に。
#                            作中では地面を掘り進むのに使用。

〇ダイタンク武装解説

#　ダイターンミサイル：　ダイターンの物と同一。機体後方から発射され、ホーミングする。
#　ダイターンウェッブ：　設定画によると、ダイターン時とは異なり、直に発射される。
#　ダイターンキャノン：　この形態では二連装がデフォになるので、威力と弾数を二連装版の
#　　　　　　　　　　  データに。また、形態面と設定面を考慮し、射程も変更しています。

〇マッハアタッカー武装解説

#　ネプチューンネット：捕獲・救出用のネット。
#　マッハバルカン：　　パトロール時のバルカンが両翼に移動した物。
#　マッハピック：　　　撒き菱。空中の敵にも当てられるが、特殊効果重視で対地攻撃に。
#　ウイングカッター：　鋭い両翼で相手を切り裂く攻撃。
#　マッハバルーン：　　風船型の爆弾。
#　マッハレーザー：　　機首から発射するレーザー。
#　マッハミサイル：　　コクピットの両側に装備されているミサイル。

〇マッパトロール武装解説

#　マッハバルカン：　　ボンネットに装備されている二連のバルカン砲。
#　マッハピック：　　　機体後方から撒かれる撒き菱。
#　溶解電波：　　　　　第三十九話でパトロールを包囲して捕獲しようとした、敵の車
#　　　　　　　　　　四台に対して流した電流？ 名称から劣属性も付けました。
#　粘着爆弾：　　　　　第十一話で排気口から発射した、粘液状の爆発物。
#　マッハバルーン：　　風船型の爆弾。

〇破嵐万丈(ユニット)武装解説

#　キャリーガン：　　　　万丈の携帯している、８連発の拳銃。対メガノイド用に強化して
#　　　　　　　　　　　ある。召喚したキャリーケースのパーツを装着する事で多種の武器
#　　　　　　　　　　　に換装可能な、多目的万能銃。
#　ダイスティック：　　　キャリーケースに収納されている、ライトセーバー状の剣。
#　キャリーマシンガン：キャリーガンのバリエーションの重機関銃。万丈の主力武装。
#　キャリーロケット：　　キャリーガンのバリエーションのバズーカ。　　
#　イオン・プラズマ砲：　第二十八話で使用した、キャリーガンのバリエーション。ダイターン
#　　　　　　　　　　　内部に侵入した、バグ軍団との戦闘に使用された、プラズマ光弾を
#　　　　　　　　　　　放つ武器。
#　怪力：　　　　　　　　ＯＰや一話等で万丈が披露してる、超人級の怪力。ボツ話に、万丈
#　　　　　　　　　　　が生身でメガボーグを投げ飛ばし、巨大な剣で両断して倒す話も
#　　　　　　　　　　　予定されていた事も反映し、気力が高まると、ガンダムファイター
#　　　　　　　　　　　並みのパワーが出せるようにしています。
