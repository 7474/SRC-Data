
# 反エンペラーコードとは？
# コミックス５巻で竜胆博士に植え付けられたバーコード皇帝に抗する能力の事です
# これを再現する為に幾つか案が出ましたで簡単ですが整理してみました

Ａ，明鏡止水等、烈の能力強化
Ｂ，バーコード皇帝に大してのみ作用する特殊能力

個人的にはＢ案が妥当かと思われますがＳＲＣの様に共闘作品ではＡ案でも良いかと思います
そして物足りなくはありますが再現する為の案を立ててみました

Ａ案
烈に"明鏡止水=反エンペラーコード, 1"若しくは任意の特殊能力を付加
初期はSerAbilityで封印し、イベントで封印解除

Ｂ案
烈に"浄解=反エンペラーコード, 1"を付加、初期はSetAbilityで封印
同じくイベントで封印を解除し下記のアイテムを烈の機体に装備
そしてバーコード皇帝には"素体"を任意のLvで設定

# 反エンペラーコード
# 反エンペラーコード, 固定, 強化パーツ
# 特殊能力
# 攻撃属性=浄
# 0, 0, 0, 0, 0


最終的な決断を下すのはシナリオ作者の方々と思われますので、
自分は指針だけを示し使用の諾否はシナリオ制作に携わる方に一任したいと思います
