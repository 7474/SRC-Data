# トレーサーシリーズの第１期量産機。
# ２０４４年にローレンス・リバモア研究所のＪ・ナッコーズらによって開発された
# 単体での大気圏突破、再突入能力を持つ汎用人型宇宙戦闘兵器です。
# 本編ラスト（ＡＤ２０５６）においては次世代型トレーサーの開発がスタートしています。
#『突撃』は、ビームサーベルとスラスターの出力を臨界ギリギリでコントロールし、巨大な
# エネルギーの刃で敵を両断する攻撃です。
# 非公式コメントによると、本来これはトレーサー運用シラバスにも存在せず
# ミカコの才能と機転で生み出された攻撃方法だそうですので、レベル条件を設定しています。
# 有る意味、ミカコがただの一兵士ではなく優れたパイロット＝ＳＦ物の主人公たらしめて
# いる要素の一つかも知れません(主役の特権とも言う^^;)
# 基本的には『レギオス』が変形しない代償としてハイスペック化…といったレベルの数値です。
# ただの量産型兵器ではなく、極めて切迫した事態に作られた機体ですので、ジェガン程
# 量産はされない、高性能＆高価格機体といった所でしょうか。
# 耐久力も、高級量産機レベルに再調整しました。
