## 伊吹純（いぶき　じゅん）　ＣＶ：滝田展明（飛田展男）
#
# 『すかいてんぷる』の店前にあるネビュラチェーンというコンビニでアルバイトをす
# る少年。その勤務態度は褒められたものではなく、日中何時間も勝手に店を閉め『す
# かいてんぷる』に通いつめている。その理由は、ある出来事から一目ぼれしたあゆに
# 少しでも多く接する（見る）ためである。
# 　護闘士に殴られても死なないタフネスと無限力を見極める力を持っていたため、あ
# ゆが女皇に覚醒する際の騒動に巻き込まれてしまう。もっとも、あゆと合法的に接近
# できるため望んで飛び込んだ、ともいえる。
# 　最後に護闘士として覚醒したが、その際に護衣がどう見ても女物であることを嘆い
# ていた。
# 　また、凱王のことをあゆの兄＝自身の義兄さんということで「兄さん」と呼ぶ。


## 大空寺あゆ（だいくうじ　あゆ）　ＣＶ：大友清里（浅井清己）
#
# 　大空寺財閥の娘にして、女皇の憑代。非常に気が強く口が悪い。ツンデレの語源と
# なったキャラクター。
# 　父親への反発などから『すかいてんぷる』でアルバイトをしているが、女皇として
# 覚醒する際に『幻朧毒蛇刻』という技を受け、酷い目に遭わされてしまう。
# 　しかし、憑代としての力と本人の強靭な意志や純の青臭さで長い時間耐え抜き、無
# 事に女皇として覚醒する。
# 　純の事を記憶に止めていない、もしくは店に来る金づる程度にしか思っていない。
# 　その名前の由来は『女皇に肖ゆ（あゆ）』から来ている。

# 　温和な性格で店員や仲間たちからの信頼も厚い。
