##############################################################################
### 国連軍 ###################################################################
#
# ・国連軍
# 　国連の保有する軍。どの国にも属さずに人類のために戦う軍とされるが、実質中身
# はアメリカ軍といっていい状況にあるらしい。
#
# ・国連軍横浜基地
# 　日本の横浜ハイヴを奪回後、オルタネイティヴ４のためにハイヴを活動させたまま
# その上に建設された基地。
# 　オルタネイティヴ計画は誘致国が人材や機材などのほとんどを賄うため、その装備
# は帝国軍のものが多く使用されている。
# 　覚悟の強い者も多く存在するが、逆に横浜基地が極東最大であることや最前線の最
# 後尾であることに安堵感を覚え、弛んでいる衛士も多く存在する。


##############################################################################
# 第207衛士訓練小隊 ##########################################################
#
# 　国連太平洋方面第11軍・横浜基地衛士訓練学校に所属する小隊。本来は涼宮茜が指
# 揮するA分隊と、榊千鶴が指揮するB分隊があった。
# 　B分隊には特殊な背景を持つ人材が集められていたため、207小隊全体に訓練兵時代
# から個室が与えられるなどの優遇措置がとられていた。
# 　ちなみに、この横浜基地衛士訓練学校はA-01の人員を育成するための機関であり、
# 卒業すると自動的にA-01へ配属となる。そのため、横浜基地衛士訓練学校へ配属にな
# るには特定の適性を満たす必要がある。


## 白銀武（しろがね　たける）　ＣＶ：相庭剛志（保志総一朗）
#
# 　国連太平洋方面第11軍・横浜基地衛士訓練学校・第207衛士訓練小隊所属の訓練兵。
# 後に正規兵として任官する。独特な機動を武器にする衛士。
# 　人類の滅亡という絶望的な未来の記憶を持ったまま、再び人類とＢＥＴＡが戦う世
# 界の10月22日に戻ってきてしまう。そこで、未来の記憶と鍛えられたままの体を使い
# 人類を救うため積極的に行動する。
# 　最初は順調に未来を変え、人類の勝利に大きく貢献する。しかし、それにより本来
# 発生しなかった様々な出来事が起こり、白銀は覚悟も決意を失い、自暴自棄になって
# "元の世界"に戻ってしまう。だが、戻った先でも白銀自身が原因となり予想外の出来
# 事が起こり、自らの死を願うほどの絶望を経験する。
# 　その後、香月教諭によりその原因が白銀を"因果導体"にしている何かのせいだと教
# えられ、二つの世界の命運を背負い、決意新たに再び戦場へと舞い戻る。

# 　その出自からいつでも全力で物事に取り組むが、それが逆に余裕のなさのあらわれ
# でもあった。しかし、伊隅大尉の助言を受け、少しずつだが力を抜いて余裕を持つこ
# とを覚えていく。

# VFA-01 #####################################################################
#
# 　オルタネイティヴ第四計画を完遂させるため、それに特化した作戦のみを遂行する
# 専任即応部隊。そのため、国連軍が関与できないような作戦であっても超法規的措置
# により派遣される。通称『伊隅戦乙女隊(伊隅ヴァルキリーズ)』。
# 　作戦内容は戦闘からＢＥＴＡの捕獲、第四計画が開発する新装備の運用評価など、
# 多岐にわたる。しかし、真の目的は00ユニットを支援し、護衛することにある。
# 　非常に損耗率が高く、発足当初は戦術機甲連隊だったが、現在では中隊規模にまで
# 縮小している。逆に、そのせいもあって生き残った隊員の能力は非常に高い。

# オルタネイティヴ計画 #######################################################
#
# 　ＢＥＴＡとのコミュニケーションを図るために始動した計画。しかし、ＢＥＴＡが
# 人類を同じ炭素系生命体でありながら生物と見なしていないため、第一〜第三計画は
# ことごとく失敗している。
# 　現在の第四計画は、それを踏まえＢＥＴＡに生物と見なされコミュニケーションを
# 取れる非炭素系擬似生命の00ユニットを開発し、ＢＥＴＡに生物として認めさせたう
# えでコミュニケーションをとろうという計画。
# 　しかし、いよいよ切羽詰ってきた人類は、この計画の失敗と同時に第五計画を発動
# させる予定となっている。
# 　ちなみにオルタネイティヴ５の概要は、全人類で選抜された約10万人を地球から脱
# 出させ、大量のＧ弾でＢＥＴＡに最終決戦を挑む、というもの。ＵＬでは『バビロン
# 作戦』、オルタネイティヴでは『トライデント作戦』と呼称された。バビロン作戦に
# はＧ弾を大量投入してＢＥＴＡに最終決戦を挑むという作戦が付随し、トライデント
# 作戦には全世界規模の反抗による陽動と凄乃皇四型を基幹とした部隊によるオリジナ
# ルハイヴを急襲するという『桜花作戦』が付随したものの、オルタネイティヴ第五計
# 画の中身は全く変わっていない。
# 　ちなみに、『バビロン作戦』によるＧ弾投下により地球に残った者たちは二年間ふ
# んばることが出来たようだが、結局のところ地球はＢＥＴＡの星になってしまう。ま
# た、その際に白銀武はＢＥＴＡによって捕らえられ、脳髄だけの状態で６０年間生存
# させられているという。
# 　これの発動は人という種が生き延びる意味ではそれなりに良策かもしれないが、実
# 質的に人類の敗北と同義である。また、危険な外宇宙への移住が確実に成功する保証
# は無い。
# 　第四計画は日本主導、第五計画は米国主導。明言されてはいないが、第三計画はソ
# ビエト主導と思われる。


## 鑑純夏（かがみ　すみか）　ＣＶ：藤原理加（田口宏子）
#
# 　どちらの世界でも武の家の隣に住む幼馴染。しかし、ＢＥＴＡが首都圏まで進攻し
# た際に捕らえられてしまい、明星作戦の時に救助されるが、その後に00ユニットに人
# 格を移植され、生物学的に死亡する。
# 　移植当初はＢＥＴＡに受けた実験や様々な出来事がＰＴＳＤとなりまともに会話も
# できないほど錯乱していたが、武と霞の献身的な付き添いの結果、最終的にはほぼ元
# の純夏と同じ状態にまで回復する。
# 　後天的に(OOユニットになって)エスパーになっており、リーディング能力とプロジ
# ェクション能力を持っている。しかし、過度のリーディングは心的負荷が大きいとさ
# れたため、つけている大きなリボンに特定の人物のリーディングを阻害する機能が付
# 与された。
#
# 　00ユニットとはオルタネイティヴ４の中核であり、機械に魂の宿った存在。開発に
# は高度な技術や高価な素材が大量に使われており、その最たるものは頭脳に用いられ
# る"量子電導脳"と呼ばれる世界最高のコンピューターで、その気になれば世界中のコ
# ンピューターをハッキングして支配下に置くことも可能。しかし、かわりにＯＤＬと
# いう特殊な液体につける必要があり、それは劣化していくため72時間以内に交換か浄
# 化する必要がある。00ユニット体内や凄乃皇にも浄化装置がついているが時間稼ぎで
# しかなく、完全な浄化には基地内の"専用の設備"が必要となる。
# 　00ユニットの能力はすさまじく、トリプルＳの国連機密情報に指定されている香月
# 副指令の年齢を、2008/06/08の香月夕子誕生日イベントでデータベースから普通に見
# つけ、情報を引き出すという離れ業を披露した。
# 　00ユニットの"00"とは"生体反応ゼロ、生物的根拠ゼロ"に由来している。

# Ｄ計画 #####################################################################
#
# 　大空寺財閥と帝国陸軍により進められる極秘計画。その全貌を知るものは少ない。
# 　Ｄ計画の内容とは、まず第一に護闘士が無自覚のまま戦場で死ぬ事を防ぐため資質
# のありそうな者を調布基地の敷地内にある“聖域（シャンバラ）”という専用施設に
# あつめることにある。そして、もう一つは護闘士が持つ無限力を通常兵器に転用する
# ことにあり、その成果として、一般人でも限定的ながら無限力を作り出せるようにす
# る装置『無限力転換炉』と護闘士の無限力を何千倍にも増幅することのできる『無限
# 力機関』、そして無限力機関を搭載した無限力応用兵器などがある。
# 　だが、実験の最中に無限力機関が暴走し、Ｄ計画に深く関わっていた者たちがほぼ
# 全滅したことで、無限力応用兵器の開発計画は事実上頓挫してしまっている。
#
# ・護闘士（さきもり）
# 　護闘士とは、天命を授かり、女皇の託宣を記した古文書『窮薩神示（きゅうりゅう
# しんじ）』を奉戴し、正義と秩序を護る者たちのことである。
# 　それぞれが守護聖獣（天命十二支、四聖、麒麟の十七体）を持ち、それにちなんだ
# 護衣（ぼうぐ）を授けられている。その使命は、女皇がこの世に在れば共に人類を繁
# 栄に導くことにあり、隠れている間は護闘士の使命を継ぎながら女皇の再降臨まで世
# の正義と秩序を守り抜くことにある。また、それとは別に四聖の護闘士には女皇が道
# を誤らぬよう監視する大任も託されている。
# 　ルーツは古代まで遡り、それぞれ女皇の居城１２方位にの門と四方の塔、その本殿
# を守るために置かれた衛士たちにある。しかし、過去には才能があるものなら護闘士
# として目覚めたものの、そのせいで逆に全滅の危機を迎えた事があった。その悲劇を
# 繰り返さぬため、女皇は持てる力の大半を使い、数こそ制限されるが確実に護闘士が

#
# ・無限力（なゆた）
# 　宇宙が混沌としていた時代は太極であり、それが陰と陽に分離しビッグバンを発生
# させた。そして、分かれた陰と陽には極限まで高めると反転し、陰は陽に、陽は陰に
# なる性質があり、それを高速回転させ混ぜ合わせることで体内に太極を作り出し、再
# び陰と陽に分けることでビッグバンを体内で発生させる。これが無限力である。
# 　無限力を使いこなす護闘士は戦術機一個大隊に相当する戦闘力を持っており、戦術
# 機に乗った場合は80時間程度の訓練と同等の効果をもたらす。

# 定数以上に増えるとハイヴを出て別のハイヴを作り始める。ハイヴの規模はフェイズ
