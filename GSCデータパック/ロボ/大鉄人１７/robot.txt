#　ブレインロボとは、巨人頭脳ブレインが作り上げた、人類抹殺用のロボット達である。

#　ＭＡＰ兵器も多いが、地上しか攻撃できないものが多い。
#　デザイン的には、機能性のみを追求したような狂ったデザインのロボットが多い。
#　しかし、１６話以降の路線変更後は、ただの狂ったロボットに成り下がってしまい、
#　非常に残念である。
#
#　参考までに、企画書には以下の名称のロボットがあったとか。
#　どの様なデザインであったのか、非常に興味深い。
#
#　　・超磁力ロボット　　　・熱線ロボット　　　・放射能ロボット

#　と言うわけである。
