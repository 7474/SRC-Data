# データ作成ガイドライン

本ドキュメントは GSC データパック準拠のデータ作成に関するガイドラインであり、生成AIへのインストラクションとしても機能します。

既存の GSC データパック（ロボ基準・拡張基準）の全データ傾向を踏まえ、一貫性のあるデータ作成を行うためのポリシーを示します。

---

## 1. 共通ポリシー（ロボ・拡張共通）

### 1.1 命名・識別子

#### パイロット名（キャラクター名）

- フルネームは全角で記述する。姓と名の間は `＝` で区切る（`・` は使わない）
  - 例: `アムロ＝レイ`, `緋村剣心`, `リナ＝インバース`
- 愛称（表示名）は全角10文字以内とする
  - 「あだ名」「苗字」「名前」のいずれかを使用する
- 同一キャラクターの時期・状態違いは `()` で区別する
  - 例: `アムロ＝レイ(0079)`, `弓さやか(後期)`

#### ユニット名

- 愛称は全角16文字以内とする
- ミドルネームは省略または頭文字を使用する
  - 例: `アーチボルト＝Ｉ＝ウィムジィ`

#### 武器名

- 全角12文字以内を推奨（最大14文字）

#### 画像識別子

- **ロボ基準**: 作品略称を使用（例: `G0079_`, `MGZ_`, `GET_`, `EVA_`）
- **拡張基準**: 3文字を基本として他と被らないものにする（例: `STF_`, `RKEN_`, `R1&2_`）
- 識別子の後にアンダースコア `_` を付け、その後にキャラクター/ユニット名をローマ字で記述する
  - パイロット画像: `{識別子}_{名前ローマ字}.bmp`（例: `G0079_AmuroRay.bmp`）
  - ユニット画像: `{識別子}_{ユニット名ローマ字}U.bmp`（例: `STF_RyuU.bmp`）
  - バリエーションは `()` で示す（例: `G0079_AmuroRay(P).bmp`）

#### 読み仮名

- ひらがなで記述する
- パイロット・ユニットともに設定する

---

### 1.2 キャラクターの格（ティア）の概念

キャラクターには作品内での「格」があり、それに応じてデータ上の実効性能に差をつける。

#### 格の分類

| 格 | 説明 | 例 |
|----|------|-----|
| 主人公格 | 作品の主人公・メインパイロット | アムロ、兜甲児、流竜馬、緋村剣心 |
| 準主役格 | 主人公に次ぐ重要キャラクター | 神隼人、セイラ＝マス、ケン＝マスターズ |
| サブキャラ格 | レギュラーだが戦闘面では主役に劣る | 弓さやか、早乙女ミチル |
| やられ役格 | 主に敵の一般兵やモブキャラ | ザク、一般兵 |

#### 格と実効性能の関係

- **主人公格は実効性能が最も高くなるようにする**
  - パイロットステータスの合計値が最も高い
  - 特殊能力の成長が早い（低レベルから高レベルに到達）
  - 性格は「強気」「超強気」が多い（気力が上がりやすく必殺技を使いやすい）
  - 搭乗ユニットも高性能（高HP・高攻撃力・高運動性の両立）

- **準主役格は特定分野で主人公に匹敵するが総合力で譲る**
  - 一部ステータスは主人公級だが、合計値はやや劣る
  - 特殊能力は持つが主人公より成長が遅い
  - ユニット性能は主人公機に準じるが何かしらの弱点がある

- **サブキャラ格は数値自体はともかく、実効性能は主要キャラクターに譲る**
  - ステータス数値は一見普通でも、SP構成が補助寄り
  - 戦闘で決定打となるSP（魂、覚醒）を持たない
  - ユニット性能が低い、または汎用機
  - 性格は「普通」が多い

- **やられ役格は明確に低性能**
  - 全ステータスが低い
  - SP・特殊能力がない場合も
  - ユニットも低性能

#### 実効性能を決める要素（重要度順）

1. ユニットの最強武器の攻撃力 × パイロットの格闘/射撃値
2. 気力上昇しやすさ（性格 × SP構成）
3. 回避/被弾のバランス（運動性 × 回避値 vs 装甲 × HP）
4. 特殊能力による底上げ（NT、底力、不屈等）

---

### 1.3 性格の設定基準

| 性格 | 気力上昇条件 | 典型的な使用対象 |
|------|-------------|----------------|
| 超強気 | 攻撃で+2, 被弾で+1, 回避で+1 | スーパー系主人公（兜甲児等） |
| 強気 | 攻撃で+1, 被弾で+1 | 主人公・戦闘好きキャラ |
| 普通 | 攻撃で+1 | 一般的なキャラクター |
| 弱気 | 被弾で-1 | 非戦闘員・子供キャラ |
| 大人しい | 被弾で-1 | 内向的・冷静なキャラ |
| 慎重 | 攻撃で+1（敵撃破時のみ） | 戦術家・参謀タイプ |

## 2. ロボ基準固有ポリシー

### 2.1 パイロットステータスの数値・範囲

パイロットのステータスは6つの基本値と性格で構成される。

```
格闘, 射撃, 命中, 回避, 技量, 反応, 性格
```

#### 数値範囲の目安

| パイロットの格 | 格闘 | 射撃 | 命中 | 回避 | 技量 | 反応 | SP |
|---------------|------|------|------|------|------|------|-----|
| 主人公格 | 155-170 | 125-165 | 135-155 | 130-165 | 168-186 | 155-170 | 50-80 |
| 準主役格 | 145-165 | 120-155 | 130-150 | 125-160 | 163-175 | 150-165 | 40-60 |
| サブキャラ格 | 120-145 | 115-140 | 125-145 | 120-145 | 155-168 | 145-160 | 40-60 |
| やられ役格 | 90-125 | 80-120 | 100-130 | 100-125 | 130-155 | 130-150 | 30-45 |

- **SP標準値は50**（全データ中最頻値）
- 主人公格のスーパー系パイロットは SP 60〜80 の高めの値をとる

#### SP構成の格による傾向（ロボ基準）

| SP | 攻撃的/決定打 | 補助的 |
|----|--------------|--------|
| 魂 | 主人公格に集中 | - |
| 覚醒 | 主人公・準主役格 | - |
| 熱血 | 主人公〜準主役格 | - |
| ひらめき | 主人公〜サブキャラ | ○ |
| 集中 | 主人公〜サブキャラ | ○ |
| 必中 | 全般 | ○ |
| 激励/応援 | - | サブキャラ格に集中 |
| 祝福/幸運 | - | サブキャラ格に多い |
| 信頼/友情 | - | サブキャラ格に多い |

### 2.2 ユニット（ロボット）ステータスの数値・範囲

ユニットのステータスは以下で構成される。

```
HP, EN, 装甲, 運動性
```

#### スーパー系ユニット

| 指標 | 典型範囲 | 備考 |
|------|---------|------|
| HP | 4,500-8,000 | 主人公機は6,000以上 |
| EN | 100-150 | |
| 装甲 | 1,200-3,000 | 主人公機は1,500以上 |
| 運動性 | 40-70 | スーパー系は低め |
| サイズ | L-M | Lが多い |

#### リアル系ユニット

| 指標 | 典型範囲 | 備考 |
|------|---------|------|
| HP | 1,600-4,000 | 主人公機は3,000前後 |
| EN | 40-100 | |
| 装甲 | 500-1,200 | リアル系は薄め |
| 運動性 | 65-120 | リアル系は高め |
| サイズ | M-S | Mが基本 |

#### 武器攻撃力の目安

| 武器種別 | 攻撃力範囲 | 備考 |
|---------|-----------|------|
| 牽制武器（バルカン等） | 700-900 | 弾数多、命中補正高 |
| 通常射撃（ライフル等） | 1,100-1,400 | 主力 |
| 格闘武器 | 1,200-1,500 | |
| 必殺技 | 1,800-2,500 | EN消費・気力制限あり |
| 最終必殺技 | 2,500-3,300 | 厳しい制限付き |
| 合体技 | 3,000-4,700 | 複数パイロット必要 |

### 2.3 ユニットとパイロットの関係性

#### 基本方針

- **パイロットは「搭乗するユニットのカテゴリ」を持つ**
  - 例: `ＭＳ`, `マジンロボ`, `ゲッターロボ`, `戦艦`
- **ユニットは「搭乗可能パイロット数」を定義する**
  - 1人乗り（標準）、複数人乗り（サブパイロット付き）
- **乗り換えオプション前提**のため、同カテゴリ内で乗り換え可能に設計する

#### 主人公機の設計指針

- 主人公パイロットの高ステータスと組み合わせて最大効果を発揮するよう設計する
- 専用機は他キャラが乗っても一定の性能を発揮するが、本来のパイロットが最も活きる構成にする
  - 例: ガンダムは高運動性＋ビーム兵器 → アムロのNT能力・高回避・高射撃で真価発揮
  - 例: マジンガーZは高HP・高装甲＋格闘必殺技 → 甲児の超強気・底力・高格闘値で真価発揮

#### 地形適応コード

ユニットの地形適応は4文字で表す（空陸水宇）。

```
A: 得意  B: 普通  C: 苦手  D: 不可  -: 移動不可
```

- 空戦型: `AA-A` (VF-1 バルキリー等)
- 陸戦型: `-ABA` (ガンダム等)
- 万能型: `AAAA` (マジンカイザー等)
- 水中型: `-ADA` (ゲッター2等)

## 3. 拡張基準固有ポリシー

### 3.1 パイロットステータスの数値・範囲

拡張基準はロボ基準と同じ6ステータス構成だが、等身大キャラクターのため一部傾向が異なる。

```
格闘, 射撃, 命中, 回避, 技量, 反応, 性格
```

#### 数値範囲の目安

| パイロットの格 | 格闘 | 射撃 | 命中 | 回避 | 技量 | 反応 | SP |
|---------------|------|------|------|------|------|------|-----|
| 主人公格 | 150-170 | 110-155 | 145-160 | 140-165 | 175-191 | 155-165 | 55-65 |
| 準主役格 | 140-160 | 105-150 | 135-155 | 130-155 | 168-180 | 150-160 | 50-60 |
| サブキャラ格 | 120-145 | 95-135 | 125-145 | 115-140 | 155-170 | 140-155 | 45-55 |
| やられ役格 | 90-120 | 75-110 | 100-125 | 90-120 | 130-155 | 120-145 | 30-45 |

- 拡張基準では射撃値よりも格闘値が重視される傾向がある（等身大格闘が多いため）

#### SP構成の格による傾向（拡張基準）

拡張基準ではロボ基準と異なり、SPの標準値は55〜65程度とやや高めの傾向がある。

| SP | 攻撃的/決定打 | 補助的 |
|----|--------------|--------|
| 熱血 | 主人公〜準主役格 | - |
| 決意/覚醒 | 主人公格に集中 | - |
| ひらめき | 主人公〜サブキャラ | ○ |
| 集中 | 主人公〜サブキャラ | ○ |
| 必中 | 全般 | ○ |
| てかげん | 全般 | ○ |
| 激励/応援 | - | サブキャラ格に集中 |

### 3.2 ユニットステータスの数値・範囲

拡張基準では「ユニット＝変身後の姿」「パイロット＝キャラクター本人」という構造が一般的。

```
HP, EN, 装甲, 運動性
```

#### 数値範囲

| パイロットの格 | HP | EN | 装甲 | 運動性 |
|---------------|-----|------|------|--------|
| 主人公格 | 5,000-7,000 | 100-160 | 1,000-1,400 | 80-100 |
| 準主役格 | 4,000-6,000 | 80-140 | 900-1,200 | 70-95 |
| サブキャラ格 | 3,000-5,000 | 60-120 | 700-1,000 | 60-85 |
| やられ役格 | 2,000-3,500 | 40-80 | 500-800 | 40-65 |

- サイズは `S`（等身大）または `M`（大型キャラ・変身体）が基本
- 拡張基準はロボ基準と比較してHP・装甲が全体的にやや低く、運動性が高い傾向

#### 武器攻撃力の目安

| 武器種別 | 攻撃力範囲 | 備考 |
|---------|-----------|------|
| 通常攻撃 | 800-1,100 | |
| 得意技 | 1,100-1,400 | |
| 必殺技 | 1,400-1,800 | EN消費・気力制限あり |
| 最終奥義 | 1,800-2,200 | 厳しい制限付き |
| 合体技 | 2,200-3,000 | 複数キャラ必要 |

### 3.3 ユニットとパイロットの関係性

#### 基本方針

拡張基準では「パイロット＝キャラクター本人」「ユニット＝戦闘形態」として設計する。

- **変身ヒーロー型**: ユニット＝変身後の姿（仮面ライダー、ウルトラマン等）
  - パイロットは変身前の人物
  - ユニットは変身後の能力体
  - 専用性が高い（乗り換え不可が多い）

- **格闘家型**: ユニット＝キャラクターの戦闘スタイル（ストリートファイター等）
  - パイロットとユニットが一体
  - ユニット名＝パイロット名
  - 完全専用

- **RPGキャラ型**: ユニット＝クラス/ジョブ（FFシリーズ等）
  - パイロットはキャラクター本人
  - ユニットはジョブや装備状態
  - やや汎用性がある場合も

#### 空適応の特殊ルール（等身大基準）

等身大基準ではロボ基準と異なり、飛行不可能ユニットでも空適応 B〜C を付与する。

これは以下の理由による:
- 飛行ユニットが限定されている
- 飛び道具を持たないユニットが多い
- 飛行ユニットとの交戦機会における不均衡の緩和

## 4. 属性・武器設計の共通基準

### 4.1 主要属性の判断基準

| 属性 | 付与基準 | 注意点 |
|------|---------|--------|
| 実 | 投擲兵器、爆発する弾頭、誘導兵器 | 砲弾・バルカンには付かない |
| Ｂ | 粒子ビーム、レーザー、光線 | ビームサーベル等の近接武器には付けない |
| 突 | 格闘攻撃（パンチ・キック等） | |
| 武 | 切り払い可能な近接武器攻撃 | |
| 格 | 射程2以上で格闘値を適用したい武器 | 射程1には付けない |
| 接 | 攻撃準備段階で相手の動きを封じてから攻撃する技（例: 超電磁スピン） | 投げ技や「防御ごと破壊する技」には付かない |
| 先 | カウンター技、牽制用武装 | |
| 連 | 連射・複数回攻撃 | レベルが高いほど命中しやすい。牽制武器は連L10（10回判定）、主力は連L5前後 |
| 貫 | 装甲貫通攻撃 | 対象は限定的 |

### 4.2 命中属性

| 属性 | 付与対象 |
|------|---------|
| Ｈ | ミサイル、ホーミング系、自律誘導射出武器 |
| 追 | AI搭載の自律判断可能な武装 |
| 誘 | パイロットの脳波/テレパシーで制御する武装 |
| 有 | 有線射出武器、ワイヤー系 |

## 5. データ作成時のチェックリスト

### 作成前確認

- [ ] 作品の主人公・準主役・サブキャラクターの格を整理したか
- [ ] 既存の同系統作品のデータを参照したか
- [ ] 識別子が他作品と重複していないか確認したか

### パイロットデータ

- [ ] ステータス値が格に応じた範囲内に収まっているか
- [ ] 主人公格のパイロットが最も高い実効性能を持つか
- [ ] SP構成が格に適切か（主人公に攻撃SP、サブキャラに補助SP）
- [ ] 性格設定がキャラクターの原作描写と一致するか
- [ ] 特殊能力のレベル成長が格に応じているか

### ユニットデータ

- [ ] HP/EN/装甲/運動性がスーパー系/リアル系の基準に合っているか
- [ ] 武器攻撃力のレンジが適切か
- [ ] 主人公機が最も実効性能が高いか
- [ ] 地形適応が原作設定と整合しているか
- [ ] サイズ設定が適切か

### 武器データ

- [ ] 属性の付与基準を満たしているか
- [ ] 命中補正のバランスが適切か（牽制武器:+20〜+30、必殺技:-10〜-30）
- [ ] EN消費/弾数/気力制限のバランスが取れているか
- [ ] 射程設定が適切か

### 命名・識別

- [ ] パイロット名・ユニット名の文字数制限を守っているか
- [ ] 画像識別子が規則に従っているか
- [ ] 読み仮名が設定されているか
